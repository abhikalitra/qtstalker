THERM


DESCRIPTION: The 'Market Thermometer' indicator is described in Dr. Alexander Elder's book Come Into My Trading Room, p.162

SYNTAX: COMMAND = THERM
STEP = <VARIABLE NAME>
HIGH = <VARIABLE NAME>
LOW = <VARIABLE NAME>
NAME = <VARIABLE NAME>
RETURNS: 'OK' = Success
'ERROR' = Failure
EXAMPLE: COMMAND=THERM;
STEP=therm;
HIGH=high;
LOW=low;
NAME=therm
