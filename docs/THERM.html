THERM - Market Thermometer

The "Market Thermometer" indicator is described in Dr. Alexander Elder's book _Come Into My Trading Room_, p.162



Script API

INDICATOR,PLUGIN,THERM,<NAME>,<SMOOTHING PERIOD>,<SMOOTHING TYPE>

  • <NAME> is the variable name given to the array
  • <SMOOTHING PERIOD> is an integer
  • <SMOOTHING TYPE> MA type
