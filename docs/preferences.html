Edit Preferences

Description:

Set and edit any system wide defaults.

Here is the edit preferences dialog. The available settings you may edit are:

  • App Font - Font used for the app.
  • Color Background - Background color used for all charts.
  • Color Border - Color used for drawing text, ticks, lines and specific areas on a chart eg. date area
  • Color Candle - Color used for Candle charts.
  • Color Down - Any down day bars located on the main chart.
  • Color Grid - Color used for the grid used on all charts.
  • Color Neutral - Any no change day bars located on the main chart.
  • Color Up - Any up day bars located on the main chart.
  • Indicator Tabs Top - Position of the indicator tabs. Checked for tabs on top. Unchecked for tabs on the bottom.
  • Navigator Left - Chart navigator appears on the left side if true, right side if false. Default is left side (true).
  • P&F Box Size - Value 0 will use the default method of box sizing.
  • P&F Reversal - Number of boxes to use for a reversal. Default is 3.
  • Paint Bar Indicator - Used only for the paint bar chart type. In order to plot, you must select an active indicator from the list, otherwise the plot will be neutral.
  • Plot Font - Font used for the charts.
