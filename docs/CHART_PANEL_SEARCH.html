CHART_PANEL_SEARCH


DESCRIPTION: Sets the symbol search filter in the main app chart panel. Updates the list on success.
SYNTAX: PLUGIN = CHART_PANEL_SEARCH
EXCHANGE = <EXCHANGE> (% symbol used for partial matches, * used for all)
SYMBOL = <SYMBOL> (% symbol used for partial matches, * used for all)
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=CHART_PANEL_SEARCH,EXCHANGE=XNYS,SYMBOL=*
