CHART_OBJECT_SELL_TEMP


DESCRIPTION: Displays a downward pointing arrow on an indicator. Used to indicate a sell point. This is a read only version that is non-interactive when displayed and is not saved to disk.
SYNTAX: PLUGIN = CHART_OBJECT_SELL_TEMP
DATE = <yyyy-MM-dd HH:mm:ss>
PRICE = <DECIMAL>
COLOR = <COLOR>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=CHART_OBJECT_SELL_TEMP,DATE=2011-02-08 16:00:00,PRICE=450.50,COLOR=red
