EXCHANGE_SEARCH_DIALOG


DESCRIPTION: Displays a dialog to search for an ISO exchange code using a city or country name as search parameters. Returns the exchange code in KEY.
SYNTAX: PLUGIN = EXCHANGE_SEARCH_DIALOG
RETURNS: '0' = Success
'ERROR' = Failure
KEY = EXCHANGE_SEARCH_DIALOG_SELECTED
EXAMPLE: PLUGIN=EXCHANGE_SEARCH_DIALOG
