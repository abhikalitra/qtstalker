AD


DESCRIPTION: Chaikin's Accumulation Distribution
SYNTAX: PLUGIN = AD
INPUT_HIGH = <VARIABLE NAME>
INPUT_LOW = <VARIABLE NAME>
INPUT_CLOSE = <VARIABLE NAME>
INPUT_VOLUME = <VARIABLE NAME>
NAME = <VARIABLE NAME>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=AD, INPUT_HIGH=high, INPUT_LOW=low, INPUT_CLOSE=close, INPUT_VOLUME=volume, NAME=ad
