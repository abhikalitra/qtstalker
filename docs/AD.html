AD - Chaikin Accumulation Distribution

There are 2 versions of this indicator available:

AD - is the standard version.
ADOSC - is an oscillator version.



Script API


AD format:

INDICATOR,PLUGIN,AD,AD,<NAME>,<PLOT TYPE>,<COLOR>
  • <NAME> is the variable name given to the AD plot
  • <PLOT TYPE> is the plot type
  • <COLOR> is the color of the plot

ADOSC format:

INDICATOR,PLUGIN,AD,ADOSC,<NAME>,<FAST PERIOD>,<SLOW PERIOD>,<PLOT TYPE>,<COLOR>

  • <NAME> is the variable name given to the ADOSC plot
  • <FAST PERIOD> is an integer for the fast value of the oscillator. Usually less than the <SLOW PERIOD>.
  • <SLOW PERIOD> is an integer for the slow value of the oscillator. Usually greater than the <FAST PERIOD>.
  • <PLOT TYPE> is the plot type
  • <COLOR> is the color of the plot
