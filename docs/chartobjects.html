Chart Objects

Description

Chart objects are basic drawing elements that are commonly used in technical analysis to mark and illustrate various chart properties. See the Chart Object Plugin  reference for details on all the chart objects used in Qtstalker.

To create a chart object, select draw mode from the main toolbar and right click with the mouse on the chart you wish to apply it to. This can be either the main chart area or any of the indicators that appear in the indicator chart area. Select "New Chart Object" from the popup menu and then select from the list the chart object to create.
