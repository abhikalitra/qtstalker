Chart Objects



Chart objects are basic drawing elements that are commonly used in technical analysis to mark and illustrate various chart properties.To create a chart object, just select an object from the vertical toolbar on the left side of the main window and follow the messages that appear in the lower left status area of the main window.

To edit, move, or delete a chart object, you must select it by first making sure the cursor is in select mode. Once selected a grab box will appear. Use this grab box to move the object or right click with the mouse to edit and delete.

Buy

A small up pointing arrow. Commonly used to indicate a buy point on a chart.


The settings are defined as follows:

  • Color - color of arrow
  • Price - the price at which the arrow is set
  • Default - if checked, all settings will default to the current values when new ones are created

Retracement

A group of horizontal line(s) used to indicate various retracement levels.

The settings are defined as follows:

  • Color - The color of the line.
  • High - The high point of the retracement range.
  • Low - The low point of the retracement range.
  • Extend - Extend lines to end of the chart.
  • Default - If checked, all settings will default to the current values when new ones are created.
  • Line 1 to Line 6 - User defined retracement levels. eg. 1.0 = 100%, 0.5 = 50%. Set to zero if no line is wanted. The deafults are fibonacci values.

HLine

A horizontal line is plotted over the entire width of the chart area. There is also optional text that appears left justified.

The settings are defined as follows:
  • Color - The color of the line. Pressing the button bring up a dialog.
  • Price - Defines the vertical price position of placement on the chart.
  • Text - Here you can enter a note which appears at the left-hand screen side.
  • Default - If checked, all settings will default to the current values when new ones are created.

Sell

A small down pointing arrow. Commonly used to indicate a sell point on a chart.

The settings are defined as follows:
  • Color - The color of the arrow.
  • Price - The price point of placement on the chart.
  • Default - If checked, all settings will default to the current values when new ones are created.

Text

An area of text placed on a chart.

The settings are defined as follows:
  • Color - The color of the text.
  • Font - The font to use for the text.
  • Text - The actual text to appear.
  • Price - The price point on the chart.
  • Default - If checked, all settings will default to the current values when new ones are created

TLine (Trend Line)

A line that can can have any angle or length the user wishes.  Commonly used to indicate trends on a chart.

The settings are defined as follows:

  • Color - The color of the trend line.
  • Default - If checked, all settings will default to the current values when new ones are created.
  • Extend - If checked, the line will extend into the infinite future. If not checked, the line will only be drawn between the start and end points selected.
  • Start Price - The first anchor point on the chart.
  • End Price - The second anchor point on the chart.

VLine

A vertical line that extends the height of the chart area.

The settings are defined as follows:
  • Color - The color of the line.
  • Default - If checked, all settings will default to the current values when new ones are created.
