Chart Objects

Chart objects are basic drawing elements that are commonly used in technical analysis to mark and illustrate various chart properties. To create a chart object, right click with the mouse on the chart you wish to place the object. The indicator context menu will be displayed. Follow the prompts in the lower left hand corner of the main window.

To edit, move, or delete a chart object, you must select it by first making sure the cursor is in select mode. Once selected a grab box will appear. Use this grab box to move the object or right click with the mouse to edit and delete.
