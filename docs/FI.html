FI - Force Index


Description:

Force Index = Volume Today * (Close Today - Close Yesterday)

Use two separate FI indicators: a very short-term EMA (2-days) and a short-term EMA (13-days).


Acknowledgements:

Alexander Elder, "Trading for a Living" ISBN: 0471592242, 0471592250



Script API


Format:

INDICATOR,PLUGIN,FI,<NAME>,<PERIOD>,<MA TYPE>
  • <NAME> is the variable name given to the plot
  • <PERIOD> an integer value used for the period
  • <MA TYPE> the MA type used
