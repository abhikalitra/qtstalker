CSV


DESCRIPTION: Imports CSV_FILE into the quote database. FORMAT is a comma delimited string of fields. DATE_FORMAT is the format to use to decipher the date field. DELIMITER is the character used to delimit the quote fields. TYPE is a user defined value e.g. 'Stock' for stock quotes, 'Futures' for futures quotes.
SYNTAX: COMMAND = CSV
STEP = <VARIABLE NAME>
CSV_FILE = <FILE_PATH>
FORMAT =
  • SYMBOL
  • NAME
  • DATE
  • OPEN
  • HIGH
  • LOW
  • CLOSE
  • VOLUME
  • OI
DATE_FORMAT = <DATE_FORMAT>
DELIMITER =
  • Comma
  • Semicolon
TYPE = <USER_DEFINED>
RETURNS: 'OK' = Success
'ERROR' = Failure
EXAMPLE: COMMAND=CSV;
STEP=csv;
CSV_FILE=/home/user/qtstalker_stuff/yahoo.csv;
FORMAT=SYMBOL,NAME,DATE,OPEN,HIGH,LOW,CLOSE,VOLUME;
DATE_FORMAT=yyyy-MM-dd;
DELIMITER=Comma;
TYPE=Stock
