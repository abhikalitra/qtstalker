MAMA - MESA Adaptive Moving Average

Description




Script API


Format:

INDICATOR,PLUGIN,MAMA,<INPUT>,<NAME MAMA>,<NAME FAMA>,<FAST LIMIT>,<SLOW LIMIT>,<MAMA PLOT TYPE>,<FAMA PLOT TYPE>,<MAMA COLOR>,<FAMA COLOR>
  • <INPUT> is an input type
  • <NAME MAMA> is the variable name given to the MAMA plot
  • <NAME FAMA> is the variable name given to the FAMA plot
  • <FAST LIMIT> an integer value
  • <SLOW LIMIT> is an integer value
  • <MAMA PLOT TYPE> is a plot type
  • <FAMA PLOT TYPE> is a plot type
  • <MAMA COLOR> is the color of the mama plot
  • <FAMA COLOR> is the color of the fama plot
