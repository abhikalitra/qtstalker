ATR - Average True Range

There are 2 versions of this indicator available:

ATR - is the standard average true range.
NATR- is a normalized version of the average true range.



Script API


ATR format:

INDICATOR,PLUGIN,ATR,ATR,<NAME>,<PERIOD>
  • <NAME> is the variable name given to the ATR plot
  • <PERIOD> is an integer period value

NATR format:

INDICATOR,PLUGIN,ATR,NATR,<NAME>,<PERIOD>

  • <NAME> is the variable name given to the ADOSC plot
  • <PERIOD> is an integer period value
