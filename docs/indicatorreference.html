Pivot Point

  • <POINT> is value between 0 and 5
    • 0 = first resistance
    • 1 = second resistance
    • 2 = third resistance
    • 3 = first support
    • 4 = second support
    • 5 = third support

Reference Dataset

INDICATOR,PLUGIN,REF,<NAME>,<INPUT>,<PERIOD>

This function creates a dataset that is <INPUT> - <PERIOD> offset. So, if you wanted to create a dataset of yesterday's close

values the string would be 'REF,yclose,Close,1' If you just wanted the today's Close then <PERIOD> would be '0'

SYMBOL

INDICATOR,PLUGIN,SYMBOL,<NAME>,<SYMBOL>,<BAR_FIELD>,<BAR_LENGTH>,<BARS>


Where <SYMBOL> is the symbol in the quote database.

<BAR_FIELD> is the bar value ie. Open, High, Low, Close, Volume, OI

<BAR_LENGTH> is the bar size. ie. 1 Minute, 5 Minute, 10 Minute, 15 Minute, 30 Minute, 60 Minute, Daily, Weekly, Monthly.

<BARS> is the number of bars to use.


Safe Zone Stop

INDICATOR,PLUGIN,SZ,<NAME>,<METHOD>,<PERIOD>,<NO_DECLINE_PERIOD>,<COEFFICIENT>

  • <METHOD> is either Long or Short
  • <PERIOD> 10 is a good default
  • <NO_DECLINE_PERIOD> 2 is good default
  • <COEFFICIENT> 2.5 is a good default
