INTEGER_DIALOG


DESCRIPTION: Displays a dialog to input an integer value. Result is stored in KEY.
SYNTAX: PLUGIN = INTEGER_DIALOG
LABEL = <USER TEXT PROPMT>
MIN = <MINIMUM INTEGER ALLOWED>
MAX = <MAXIMUM INTEGER ALLOWED>
VALUE = <DEFAULT VALUE>
RETURNS: '0' = Success
'ERROR' = Failure
KEY = INTEGER_DIALOG_INTEGER
EXAMPLE: PLUGIN=INTEGER_DIALOG,LABEL=Enter period,MIN=1,MAX=99,VALUE=10
