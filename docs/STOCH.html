STOCH - Fast Stochastics

Description



Script API


INDICATOR,PLUGIN,STOCH,<NAME FASTK>,<NAME FASTD>,<FASTK PERIOD>,<FASTD PERIOD>,<FASTD MA TYPE>,<FASTK PLOT TYPE>,<FASTD PLOT TYPE>,<FASTK COLOR>,<FASTD COLOR>
  • <NAME FASTK> is the variable name given to the array
  • <NAME FASTD> is the variable name given to the array
  • <FASTK PERIOD> is an integer
  • <FASTD PERIOD> is an integer
  • <FASTD MA TYPE> MA type
  • <FASTK PLOT TYPE> is a plot type
  • <FASTD PLOT TYPE> is a plot type
  • <FASTK COLOR> is the color of the plot
  • <FASTD COLOR> is the color of the plot
