GROUP_DATABASE_LOAD


DESCRIPTION: Returns a semicolon delimited string of the group contents of NAME. The string is stored in KEY.
SYNTAX: PLUGIN = GROUP_DATABASE_LOAD
NAME = <GROUP NAME>
RETURNS: '0' = Success
'ERROR' = Failure
KEY = GROUP_DATABASE_LOAD_GROUP
EXAMPLE: PLUGIN=GROUP_DATABASE_LOAD,NAME=group1
