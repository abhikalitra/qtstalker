QuoteServer API - YahooHistorical


These script commands allow quote access to yahoo financial.




YahooHistorical,H,<START DATE>,<END DATE>,<SYMBOL>\n

Downloads historical quotes for the yahoo symbol <SYMBOL>.

  • <START DATE> is the start date and time of the first bar returned. the format is YYYYMMDD eg. '20100101'
  • <END DATE> is the end date up to and including the last barreturned. the format is YYYYMMDD eg. '20101231'
  • <SYMBOL> is the yahoo symbol eg. 'GOOG'
Success returns 'OK\n'. Failure returns 'ERROR\n'.

A full example would be to download GOOG quotes 'YahooHistorical,H,20100104,20101231,GOOG\n'




YahooHistorical,D,<YAHOO CODE>,<DETAIL>,<SYMBOL>\n

Downloads the yahoo finance code data for the provided yahoo symbol.

  • <YAHOO CODE> the yahoo finance data code(s) delimited by a colon. eg. 'x:n' Some yahoo codes follows:
    • n - company name
    • x - exchange
  • <DETAIL> the detail code(s) delimited by a colon eg. 'Name:T1'
  • <SYMBOL> is the yahoo symbol eg. 'GOOG'
Success returns 'OK\n'. Failure returns 'ERROR\n'.

A full example would be to request company name and exchange data 'YahooHistorical,D,n:x,Name:T1,GOOG\n'
