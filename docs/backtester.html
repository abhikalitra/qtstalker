Back Tester

The back tester allows you to create trading rules using indicators to buy and sell based on signals the indicators emit. You can then run "what if" scenarios and then display the results in detail. Overall, you can use the back tetser to test your ideas for profitability. The back tester dialog contains several pages of options you use to create and fine tune your trading rules. The page options are as follows:

Rules - This is where you select the indicators to be used.
Stops - This is where you select the stops the rule will use when trading.
Testing - The finer details that apply to a particular test such as margins, account balance etc.
Reports - This is where the trading summary will be found along with each trade detail that apply to the most recent test.
Charts - Here is where you will find two charts that apply to the most recent test.
