MAVP - Moving Average with Variable Period

Description




Script API

INDICATOR,PLUGIN,MAVP,<NAME>,<INPUT>,<INPUT 2>,<MIN PERIOD>,<MAX PERIOD>,<MA TYPE>

  • <NAME> is the variable name given to the plot
  • <INPUT> is an input type
  • <INPUT 2> is an input type
  • <MIN PERIOD> is an integer value
  • <MAX PERIOD> an integer value
  • <MA TYPE>  MA type
