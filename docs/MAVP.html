MAVP


DESCRIPTION: Moving Average Variable Period
SYNTAX: PLUGIN = MAVP
NAME = <VARIABLE NAME>
INPUT = <VARIABLE NAME>
INPUT2 = <VARIABLE NAME>
PERIOD_MIN = <INTEGER>
PERIOD_MAX = <INTEGER>
MA_TYPE = <MA TYPE>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=MAVP,NAME=mavp,INPUT=close,INPUT2=close,PERIOD_MIN=2,PERIOD_MAX=30,MA_TYPE=EMA
