CONTROL_PANEL


DESCRIPTION: Issues control commands to the main app control panel.
SYNTAX: PLUGIN = CONTROL_PANEL
METHOD = <>
  • CHART_REFRESH - refreshes the chart panel
  • GROUP_REFRESH - refreshes the group panel
  • SCRIPT_RUN - runs a script from the script panel
    • NAME = script name to run
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=CONTROL_PANEL, METHOD=CHART_REFRESH

PLUGIN=CONTROL_PANEL, METHOD=SCRIPT_RUN, NAME=fooscript
