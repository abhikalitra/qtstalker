INDICATOR_PLOT_VALUE_SET


DESCRIPTION: Sets the VALUE and COLOR of NAME at INDEX position.
SYNTAX: PLUGIN = INDICATOR_PLOT_VALUE_SET
NAME = <VARIABLE NAME>
VALUE = <DECIMAL>
INDEX = <INTEGER>
COLOR = <COLOR> - optional, default = red
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=INDICATOR_PLOT_VALUE_SET,NAME=close,VALUE=100,COLOR=red
