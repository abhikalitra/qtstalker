MACD - Moving Average Convergence Divergence


Description



Script API


Format:

INDICATOR,PLUGIN,MACDEXT,<INPUT>,<NAME MACD>,<NAME SIGNAL>,<NAME HIST>,<FAST PERIOD>,<FAST MA TYPE>,<SLOW PERIOD>,<SLOW MA TYPE>,<SIGNAL PERIOD>,<SIGNAL MA TYPE>

  • <INPUT> is an input type
  • <NAME MACD> is the variable name given to the MACD plot
  • <NAME SIGNAL> is the variable name given to the signal plot
  • <NAME HIST> is the variable name given to the histogram plot
  • <FAST PERIOD> an integer value used for the fast period
  • <FAST MA TYPE> the MA type used for the fast MA
  • <SLOW PERIOD> an integer value used for the slow period
  • <SLOW MA TYPE> the MA type used for the slow MA
  • <SIGNAL PERIOD> an integer value used for the signal period
  • <SIGNAL MA TYPE> the MA type used for the signal MA
