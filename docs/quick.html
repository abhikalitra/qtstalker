Getting Started: A Quick Howto

  • When Qtstalker first starts, there will be some default indicators. However there will not be any data to display.
  • Get some quick data from the Yahoo service. Select the Script panel. From its background context menu (i.e. right-click), select "Run Sript". There is a pre-installed script for Yahoo data.
  • There are some other data loading example scripts in the "/usr/local/share/qtstalker/quote/" directory. Use these to load some end-of-day data using "Comma-Separated Values" text files. Perhaps you already have an EOD data supplier.
  • To load a chart, select the "Charts" button in the Navigation panel. Select the "All Symbols" button or use the "Symbol Search" to narrow the list. Navigate the list and select a symbol to load by double-clicking it.
  • Some default indicators were created when you installed Qtstalker.
  • To add more indicators, open the "New Indicator" dialog from the chart background context menu (i.e. right-click) and specify an indicator. There is a set of example scripts available in the "/usr/local/share/qtstalker/indicator/" directory.
  • Use the Script API Reference to create your own indicators, data loading, and other facilities. There is a set of example scripts available at "/usr/local/share/qtstalker/" and some others in the "misc/CUS_examples/" directory.
  • This should be enough to get you started for now. Consult the documentation and play with the functions to become familiar with how Qtstalker works.
