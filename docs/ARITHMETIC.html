ARITHMETIC


DESCRIPTION: Applies arithmetic to INPUT and INPUT2 depending on METHOD.
SYNTAX: PLUGIN = ARITHMETIC
NAME = <VARIABLE NAME>
INPUT = <VARIABLE NAME>
INPUT2 = <VARIABLE NAME>
METHOD = <>
  • ADD - adds INPUT and INPUT2
  • DIV - divides INPUT by INPUT2
  • MULT - multiplies INPUT by INPUT2
  • SUB - subtracts INPUT2 from INPUT
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=ARITHMETIC, METHOD=ADD,NAME=add,INPUT=Close,INPUT2=Close
