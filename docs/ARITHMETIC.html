ARITHMETIC


DESCRIPTION: Applies arithmetic to INPUT and INPUT2 depending on METHOD.
SYNTAX: COMMAND = ARITHMETIC
STEP = <VARIABLE NAME>
NAME = <VARIABLE NAME>
INPUT = <VARIABLE NAME>
INPUT_2 = <VARIABLE NAME>
METHOD =
  • ADD - adds INPUT and INPUT_2
  • DIV - divides INPUT by INPUT_2
  • MULT - multiplies INPUT by INPUT_2
  • SUB - subtracts INPUT_2 from INPUT
RETURNS: 'OK' = Success
'ERROR' = Failure
EXAMPLE: COMMAND=ARITHMETIC; STEP=add; METHOD=ADD; NAME=add; INPUT=Close; INPUT_2=Close
