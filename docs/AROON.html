AROON - Aroon Indicator


DESCRIPTION:    Aroon

SYNTAX:             PLUGIN = AROON
                           NAME_UPPER = <VARIABLE NAME>
                           NAME_LOWER = <VARIABLE NAME>
                           PERIOD = <INTEGER>

RETURNS:           '0' = Success
                          'ERROR' = Failure

EXAMPLE:           PLUGIN=AROON,NAME_UPPER=aroonup,NAME_LOWER=aroondown,PERIOD=10
