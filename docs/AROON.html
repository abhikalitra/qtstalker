AROON - AROON

There are 2 versions of this indicator available:

AROON - is the standard version.
AROONOSC - is an oscillator version.



Script API


AROON format:

INDICATOR,PLUGIN,AROON,AROON,<UP NAME>,<DOWN NAME><PERIOD>,<UP PLOT TYPE>,<DOWN PLOT TYPE>,<UP COLOR>,<DOWN COLOR>
  • <UP NAME> is the variable name given to the up band
  • <DOWN NAME> is the variable name given to the down band
  • <PERIOD> is an integer
  • <UP PLOT TYPE> is the up band plot type
  • <DOWN PLOT TYPE> is the down band plot type
  • <UP COLOR> is the color of the up band
  • <DOWN COLOR> is the color of the down band


OSC format:

INDICATOR,PLUGIN,AROON,OSC,<NAME>,<FAST PERIOD>,<SLOW PERIOD>,<PLOT TYPE>,<COLOR>

  • <NAME> is the variable name given to the OSC plot
  • <FAST PERIOD> is an integer for the fast value of the oscillator. Usually less than the <SLOW PERIOD>.
  • <SLOW PERIOD> is an integer for the slow value of the oscillator. Usually greater than the <FAST PERIOD>.
  • <PLOT TYPE> is the plot type
  • <COLOR> is the color of the plot
