Chart Panel

Description:

The chart tab allows you to view and serach for charts in the quotes database. Double clicking with the mouse will display the selected chart. At the top of the list there are some buttons. The "All Symbols" button will list all possible charts in the database. The "Symbol Search" button will search and display any charts with a filter string. Either a literal string like "IBM" would search for an exact match. A string like "%BM%" woulld search for any charts that contain "BM" in the symbol name. The % character is used for globbing.

Right clicking with the mouse on the chart list will display the context menu. The menu has the following functions.

Add To Group

This function will display a dialog that allows you to select a group where the chart is to be added.


All Symbols

Lists all charts in the quotes database.


Symbol Search

Allows user to search for specific charts using a string filter.
