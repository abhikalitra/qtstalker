STOCK_SPLIT


DESCRIPTION: Will adjust all values for EXCHANGE:SYMBOL in the database before DATE to the new SPLIT ratio.
SYNTAX: PLUGIN = STOCK_SPLIT
EXCHANGE = <EXCHANGE CODE>
SYMBOL = <TICKER>
DATE = <DATE> - ISO date format (yyyy-MM-dd HH:mm:ss)
SPLIT = <SPLIT RATIO> - e.g. 2:1, 3:2, 1:10
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=STOCK_SPLIT, EXCHANGE=XNYS, SYMBOL=GOOG, DATE=2011-03-15 16:00:00,SPLIT=2:1
