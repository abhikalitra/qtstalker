ADXR


DESCRIPTION: Average Directional Movement Index Rating

ADXR is an ADX that takes the current ADX level, adds the ADX level PERIOD number of days ago and divides it by two. This results in a smoother ADX reducing whipsaws.
SYNTAX: PLUGIN = ADXR
INPUT_HIGH = <VARIABLE NAME>
INPUT_LOW = <VARIABLE NAME>
INPUT_CLOSE = <VARIABLE NAME>
NAME = <VARIABLE NAME>
PERIOD = <INTEGER> - default = 14
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=ADXR, INPUT_HIGH=high, INPUT_LOW=low, INPUT_CLOSE=close, NAME=adxr, PERIOD=10
