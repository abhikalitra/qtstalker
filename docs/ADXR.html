ADXR - Average Directional Movement Index Rating


DESCRIPTION:    ADXR is an ADX that takes the current ADX level, adds the ADX level PERIOD number of days ago and divides it by two. This results in a smoother ADX reducing whipsaws.

SYNTAX:             PLUGIN = ADXR
                           NAME = <VARIABLE NAME>
                           PERIOD = <INTEGER>

RETURNS:           '0' = Success
                          'ERROR' = Failure

EXAMPLE:           PLUGIN=ADXR,NAME=adxr,PERIOD=10
