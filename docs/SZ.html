SZ


DESCRIPTION: The SafeZone Stop indicator is described in Dr. Alexander Elder's book Come Into My Trading Room, p.173
SYNTAX: PLUGIN = SZ
INPUT_HIGH = <VARIABLE NAME>
INPUT_LOW = <VARIABLE NAME>
NAME = <VARIABLE NAME>
METHOD = <TYPE>
  1. LONG
  2. SHORT
PERIOD = <INTEGER>
NO_DECLINE_PERIOD = <INTEGER>
COEFFICIENT = <DECIMAL>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=SZ, INPUT_HIGH=high, INPUT_LOW=low, NAME=sz, METHOD=SHORT, PERIOD=10, NO_DECLINE_PERIOD=2, COEFFICIENT=2
