SZ - Safe Zone Stop

Description



Script API


INDICATOR,PLUGIN,SZ,<NAME>,<METHOD>,<PERIOD>,<NO DECLINE PERIOD>,<COEFFICIENT>

  • <NAME> is the variable name given to the array
  • <METHOD> is:
    • Long
    • Short
  • <PERIOD> is an integer (10 is a good default)
  • <NO DECLINE PERIOD> is an integer (2 is good default)
  • <COEFFICIENT> is a decimal value (2.5 is a good default)
