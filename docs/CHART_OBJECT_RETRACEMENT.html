CHART_OBJECT_RETRACEMENT


DESCRIPTION: Creates a multi level retracement chart object.

SYNTAX: COMMAND = CHART_OBJECT_RETRACEMENT
STEP = <VARIABLE NAME>
DATE = <yyyy-MM-dd HH:mm:ss>
DATE_2 = <yyyy-MM-dd HH:mm:ss>
COLOR = <COLOR>
HIGH = <DECIMAL>
LOW = <DECIMAL>
LINE_1 = <DECIMAL>
LINE_2 = <DECIMAL>
LINE_3 = <DECIMAL>
LINE_4 = <DECIMAL>
LINE_5 = <DECIMAL>
LINE_6 = <DECIMAL>
Z = <INTEGER>
RETURNS: 'OK' = Success
'ERROR' = Failure
EXAMPLE: COMMAND=CHART_OBJECT_RETRACEMENT;
STEP=retracement1;
DATE=2011-01-01 09:30:00;
DATE_2=2011-02-08 16:00:00;
COLOR=red;
HIGH=500;
LOW=400;
LINE_1=0.236;
LINE_2=0.381;
LINE3=0.618;
Z=1
