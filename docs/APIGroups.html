Scripting API - Groups

ADD

GROUP,ADD,<GROUP NAME>,<EXCHANGE>,<SYMBOL>

This function will add an a security symbol item to a Qtstalker group. If <GROUP NAME> does not exist, it will be created and the symbol added.

  • <GROUP NAME> is the name of the group to add items to.
  • <EXCHANGE> is the exchange the symbol trades on.
  • <SYMBOL> is the symbol of the item we want to add to the group.

On success a '0' is returned. On failure, a 'ERROR' is returned.

DELETE

GROUP,DELETE,<GROUP NAME>

This function will delete a Qtstalker group and all of its contents.

  • <GROUPNAME> is the name of the group to delete.

On success a '0' is returned. On failure, a 'ERROR' is returned.

GET

GROUP,GET,<GROUP NAME>

This function will return a csv list of <GROUP NAME> contents.

  • <GROUP NAME> is the name of the group.

On success a csv string is made up of exchange,symbol pairs is returned. On failure, the string "ERROR" is returned.
