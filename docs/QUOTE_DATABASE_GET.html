QUOTE_DATABASE GET


DESCRIPTION: Load quotes from the database.
SYNTAX: PLUGIN = QUOTE_DATABASE
METHOD = GET
NAME_DATE = <VARIABLE NAME> - Optional use
NAME_OPEN = <VARIABLE NAME> - Optional use
NAME_HIGH = <VARIABLE NAME> - Optional use
NAME_LOW = <VARIABLE NAME> - Optional use
NAME_CLOSE = <VARIABLE NAME> - Optional use
NAME_VOLUME = <VARIABLE NAME> - Optional use
NAME_OI = <VARIABLE NAME> - Optional use
EXCHANGE = <EXCHANGE>
SYMBOL = <SYMBOL>
LENGTH = <BAR LENGTH>
RANGE = <DATE RANGE>
RETURNS: '0' = Success
'ERROR' = Failure
KEY = QUOTE_DATABASE_GET_NAME - returns the company name
EXAMPLE: PLUGIN=QUOTE_DATABASE, METHOD=GET, NAME_CLOSE=close, NAME_VOLUME=volume, EXCHANGE=XNYS, SYMBOL=GOOG, LENGTH=D, RANGE=1 Year
