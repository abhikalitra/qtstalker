Install Plugins

Description

Use this function to install Qtstalker plugins. Only plugins that are installed using this function will work. Currently, here are the types of plugins available:

  • Indicator - These are the indicators used like RSI, MACD etc.
  • Quote - These are used for downloading or importing different kinds of quote data into Qtstalker.

When this function is selected, you will be presented with a file dialog. Use this dialog to navigate to where you have compiled the plugin. The plugin must have a file descriptor of *.so to be recognized. Any plugins that have a duplicate name will be overwritten.
