NEW_DIALOG


DESCRIPTION: Displays a dialog which a new text item can be edited. The new text string is stored in KEY.
SYNTAX: PLUGIN = NEW_DIALOG
TITLE = <TEXT>
ITEMS = <semicolon delimited string of items to avoid>
RETURNS: '0' = Success
'ERROR' = Failure
KEY = NEW_DIALOG_NAME
EXAMPLE: PLUGIN=NEW_DIALOG,TITLE=New Indicator,ITEMS=indicator1;indicator2
