TA-Lib

Qtstalker uses a separate package of indicators called TA-Lib : Technical Analysis Library.

For all documentation, see the tadoc.org Function Index and follow the links to the TA-Lib implementation source code. Each function, and the input parameters and output values, are described in its code. You also have a local copy wherever you installed the TA-Lib source. See the complete list (which includes the candlestick functions) in the TA-Lib source control system.

See TA-Lib : Function List which lists every function, with its short name and its long name.

For development information, see the explanation of the Abstraction Layer.

For more detailed information regarding TA-Lib, please consult the TA-Lib website.
