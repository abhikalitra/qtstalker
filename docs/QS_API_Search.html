QuoteServer API - Search

These script commands searches the database using the exchange and symbol fields on partial and/or full matches.

S

Search,S,<EXCHANGE>,<SYMBOL>\n

Searches the database on provided <EXCHANGE> and <SYMBOL> fields and returns the symbols that match.

  • <EXCHANGE> is the exchange the quote originates from. eg. 'XNYS' or for partial matches use % eg. '%NYS' or '%NY%'
  • <SYMBOL> is the symbol eg. 'GOOG' or for partial matches use % eg. '%OOG' or '%OO%'

Success returns a string of '<EXCHANGE>,<SYMBOL>' pairs. Each pair is delimited by a colon. End of data is delimited by a \n. Failure returns 'ERROR\n'.

A full example would be to return any exchange,symbol pairs that end in NYS,OOG 'Search,S,%NYS,%OOG\n'

E

Search,E\n

Searches the database and returns all exchange names found. No duplicates are returned.

Success returns a string of exchanges delimited by a colon. End of data is delimited by a \n. Failure returns 'ERROR\n'.
