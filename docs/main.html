Qtstalker GUI

From top to bottom:

Titlebar

The title bar displays the current chart loaded. The format is company name, chart symbol enclosed in brackets, the current compression format.

Menubar

Toolbar

The most used Qtstalker functions are found here in the toolbar. From left to right they are:

  • Quit Qtstalker - shuts down the application.
  • Preferences - displays the application preferences dialog
  • Side panel toggle - Shows/hides the side panel area.
  • Grid toggle - Shows/hides the chart grids.
  • Scale to screen - Toggles the scaling of charts to visible data or entire data series.
  • Crosshairs - Toggles the chart crosshairs. If the crosshairs are turned on; left clicking the mouse anywhere on the main chart, a x/y pair of intersecting lines will be drawn at the point where the mouse was clicked. This allows you to locate specific values on the main chart. The value will appear in the status area. The status message will be the x(date) and y(value) of the pointer.
  • Draw Mode - Toggle draw mode for chart object functions.
  • Refresh Chart - Refreshes chart every X minutes. Minutes is set in the preferences dialog. This allows intra day charts to be updated if the quotes database is being updated frequently.
  • New Indicator - Displays the new indicator dialog.
  • Data window - Displays the data window.
  • Help - Displays the help system dialog.
  • Compression - Toggles the chart compression. eg.daily or weekly bars.
  • Compression Buttons - The buttons are user defined to display. They are simply shortcuts to the different compression options in the compression list. The buttons are set in the preferences dialog.
  • Zoom In - If enabled, will advance through zoom selections the user has made in the current chart.
  • Zoom Out - If enabled, will backstep through zoom selections the user has made in the current chart.
  • Bar spacing quick change buttons - Change the pixel space between each bar as you preset quickly. 2 buttons are available.
  • Bars - Loads the total number of bars at one time.
  • Recent Charts - Keeps a list of charts displayed.


Indicator Tabs

Up to three rows of indicators can be displayed. Each row can contain many indicators that are tabbed for quick display. Any indicator created will create it's own chart on the requested tab row. Right clicking with the mouse will bring up the chart pop up context menu with the following functions.

  • New Chart Object - Add a new chart object to the selected indicator.
  • Delete All Chart Objects - Permanently remove all chart objects from the selected indicator.
  • Date - Toggle the date area of the indicator off/on
  • Log Scaling - Toggle log scaling of the selected indicator off/on.

Statusbar

The status bar, at the bottom of the screen, is used to display status messages during various operations and interactions with the user. The status bar will also display the results of the cross-hairs when you click on either window. The format of the chart pointer is 2 values. First is the x value or date and the second is the y value or the plotted value.

Side Panel

Left or right of the indicators is the side panel. The chart, groups, indicator panel tabs. These tabs give you acess to assorted functions that apply to each.

The chart tab allows you to view and serach for charts in the quotes database. Double clicking with the mouse will display the selected chart. At the top of the list there are some buttons. The "All Symbols" button will list all possible charts in the database. The "Symbol Search" button will search and display any charts with a filter string. Either a literal string like "IBM" would search for an exact match. A string like "%BM%" woulld search for any charts that contain "BM" in the symbol name. The % character is used for globbing.

Right clicking with the mouse on the chart list will display the context menu. The menu has the following functions.

  • Add To Group - This function will display a dialog that allows you to select a group where the chart is to be added.
  • All Symbols - Lists all charts in the quotes database.
  • Symbol Search - Allows user to search for specific charts using a string filter.

The group tab allows you to create, edit and delete groups. A group is simply a collection of charts defined by the user. Above the groups is a combo box that contains a list of all groups. Use this drop down list to switch between groups. Right clicking with the mouse in the list will display the context menu. The functions are as follows.

  • New Group - Here you can create a new group. Selecting this will bring up a dialog where you are asked for a name to call your new group. The name must be unique, if not an error message will display.
  • Add To Group - Displays a dialog that allows you to add the selected group item to the group of your choice.
  • Delete Group Items - Opens a dialog that allows you to select one or more charts to remove from the group.
  • Delete Group - Deletes the current group and all of its contents.

This panel is used to manage indicators. At the top of the panel are some buttons. The "Show active indicators" button will display all indicators that have been turned on. The "Show all indicators button" will list all indicators in the database. The "Search" button will search for a specific indicator(s) either an exact match or globbing using the % character for a partial match. If you want to flip on/off an individual indicator, just double click on the indicator in the list. The icon in the front of the list item displays the status of the indicator. A green check mark means 'on' and a red stop sign means 'off'.

Right clicking with the mouse on the list will display the context menu. It has the following functions.

  • New Indicator - Will display the New Indicator dialog. Allows the user to add a new indicator to the database. When added, the new indicator dialog will be displayed and turned on.
  • Edit Indicator - Allows the user to edit the parameters of the selected indicator.
  • Delete Indicator - Allows the user to delete selected indicator.

The data panel, at the bottom-left/bottom right of the screen, is used to display the data values as the mouse cursor moves over the chart or indicator windows. When the cursor is in the chart window (left-hand screenshot), the panel will show the OHLC price information, the date, and the values for all chart indicators. When the cursor is in the indicator window (right-hand screenshot), the panel will show the date, and the values for the active indicator.
