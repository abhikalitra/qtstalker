Main Screen

The figure below illustrates the what a typical Qtstalker app looks like. The basic parts of the app going from top to bottom are:

  1. The title bar displays the current chart loaded.
  2. The menu bar.
  3. The tool bar area consists of access to the most used charting functions like chart types, bar spacing, grid, bars etc.
  4. The chart, groups, indicator, portfolio and backtesting and scanner panel buttons. These buttons give you access to assorted functions that apply to each.
  5. Main chart. This is the main plot for Open, High, Low, Close data for every chart. Any indicators that apply to OHLC data will also be plotted here. eg Bollinger Bands, MA's etc.
  6. Any stacked indicators are located under the main plot.
  7. Indicator tabbar. Any indicators that don't apply to the main and not stacked will be plotted here. eg. MACD, RSI etc.
