Williams Percent R - WILLR


Parameters:

  • Color - WILLR color
  • Line Type - WILLR line type
  • Label - The text to identify the WILLR plot
  • Period - WILLR period

Description:

The value is graphed on a 0 to -100 scale (historically Williams Percent R was graphed on a scale from 0 to 100 with 0 at the top and 100 at the bottom.)
