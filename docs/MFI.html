MFI - Money Flow Index

Description



Script API

INDICATOR,PLUGIN,MFI,<NAME>,<PERIOD>,<SMOOTHING PERIOD>,<SMOOTHING TYPE>,<PLOT TYPE>,<COLOR>

  • <NAME> is the variable name given to the plot
  • <PERIOD> is an integer
  • <SMOOTHING PERIOD> is an integer
  • <SMOOTHING TYPE> MA type
  • <PLOT TYPE> is a plot type
  • <COLOR> is the color of the plot
