Custom Indicator Types

Moving Average Types
  • EMA - Exponential
  • SMA - Simple
  • WMA - Weighted
  • Wilder - Wilder
Line Types
  • Dot
  • Dash
  • Line
  • Histogram
  • Histogram Bar
Operator Types
  • EQ - Equals
  • LT - Less than
  • LTEQ - Less than or equal to
  • GT - Greater than
  • GTEQ - Greater than or equal to
  • AND - And comparison
  • OR - Or comparison
Data Array Types
  • Data1 - This is a data array of values that refers to a previous function in a CUS formula. ie 1 refering to line 1 in the formula.
  • Data2 - This is a data array of values. It allows 2 different values. It behaves exactly like data1 as well as being able to use a constant value. ie #10.375. To specify a value constant use the # sign to distinguish between a constant value and a formula line number.
  • Data3 - Same as Data2
  • Data4 - Same as Data2
  • Data5 - Same as Data2
