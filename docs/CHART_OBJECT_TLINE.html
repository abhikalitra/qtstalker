CHART_OBJECT_TLINE


DESCRIPTION: Displays a trend line on an indicator.
SYNTAX: COMMAND = CHART_OBJECT_TLINE
STEP = <VARIABLE NAME>
DATE = <yyyy-MM-dd HH:mm:ss>
DATE_2 = <yyyy-MM-dd HH:mm:ss>
PRICE = <DECIMAL>
PRICE_2 = <DECIMAL>
COLOR = <COLOR>
Z = <INTEGER>
RETURNS: 'OK' = Success
'ERROR' = Failure
EXAMPLE: COMMAND=CHART_OBJECT_TLINE;
STEP=tline1;
DATE=2011-01-01 09:30:00;
DATE_2=2011-02-08 16:00:00;
PRICE=450.50;
PRICE_2=550.5;
COLOR=red;
Z=1
