CHART_OBJECT_TLINE


DESCRIPTION: Displays a trend line on an indicator.
SYNTAX: PLUGIN = CHART_OBJECT_TLINE
NAME = <VARIABLE NAME>
INDICATOR = <INDICATOR NAME>
EXCHANGE = <EXCHANGE>
SYMBOL = <SYMBOL>
DATE = <yyyy-MM-dd HH:mm:ss>
DATE2 = <yyyy-MM-dd HH:mm:ss>
PRICE = <DECIMAL>
PRICE2 = <DECIMAL>
COLOR = <COLOR>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=CHART_OBJECT_TLINE,NAME=tline1,INDICATOR=Bars,EXCHANGE=XNYS,SYMBOL=GOOG,DATE=2011-01-01 09:30:00,DATE2=2011-02-08 16:00:00,PRICE=450.50,PRICE2=550.5,COLOR=red
