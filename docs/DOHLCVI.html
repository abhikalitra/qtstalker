DOHLCVI


DESCRIPTION: Creates an array of METHOD values from the currently loaded quote bars.
SYNTAX: PLUGIN = DOHLCVI
NAME = <VARIABLE NAME>
METHOD =
  • D - Date
  • O - Open
  • H - High
  • L - Low
  • C - Close
  • V - Volume
  • I - Open Interest
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=DOHLCVI,METHOD=C,NAME=close
