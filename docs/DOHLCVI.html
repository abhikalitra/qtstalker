DOHLCVI


DESCRIPTION: Creates arrays of bar values from the currently loaded quote bars. All NAME_* parms are optional, so only the NAME_* parms used will be created.
SYNTAX: PLUGIN = DOHLCVI
NAME_DATE = <VARIABLE NAME>
NAME_OPEN = <VARIABLE NAME>
NAME_HIGH = <VARIABLE NAME>
NAME_LOW = <VARIABLE NAME>
NAME_CLOSE = <VARIABLE NAME>
NAME_VOLUME = <VARIABLE NAME>
NAME_OI = <VARIABLE NAME>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=DOHLCVI, NAME_DATE=date, NAME_CLOSE=close
