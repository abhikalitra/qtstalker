HT - Hilbert Transform


There are 6 different HT methods:

HT DCPERIOD
HT DCPHASE
HT PHASOR
HT SINE
HT TRENDLINE
HT TRENDMODE



Script API

There are 6 different methods.


Format:

INDICATOR,PLUGIN,HT,DCPERIOD,<NAME>,<INPUT>,<PLOT TYPE>,<COLOR>
  • <NAME> is the variable name given to the plot
  • <INPUT> is an input type array
  • <PLOT TYPE> is the plot type
  • <COLOR> is the color of the plot


INDICATOR,PLUGIN,HT,DCPHASE,<NAME>,<INPUT>,<PLOT TYPE>,<COLOR>

  • <NAME> is the variable name given to the plot
  • <INPUT> is an input type array
  • <PLOT TYPE> is the plot type
  • <COLOR> is the color of the plot

INDICATOR,PLUGIN,HT,TRENDLINE,<NAME>,<INPUT>,<PLOT TYPE>,<COLOR>
  • <NAME> is the variable name given to the plot
  • <INPUT> is an input type array
  • <PLOT TYPE> is the plot type
  • <COLOR> is the color of the plot

INDICATOR,PLUGIN,HT,TRENDMODE,<NAME>,<INPUT>,<PLOT TYPE>,<COLOR>
  • <NAME> is the variable name given to the plot
  • <INPUT> is an input type array
  • <PLOT TYPE> is the plot type
  • <COLOR> is the color of the plot


INDICATOR,PLUGIN,HT,PHASE,<INPUT>,<PHASE NAME>,<QUAD NAME>,<PHASE PLOT TYPE>,<QUAD PLOT TYPE>,<PHASE COLOR>,<QUAD COLOR>
  • <INPUT> is an input type array
  • <PHASE NAME> is the variable name given to the phase plot
  • <QUAD NAME> is the variable name given to the quad plot
  • <PHASE PLOT TYPE> is the phase plot type
  • <QUAD PLOT TYPE> is the quad plot type
  • <PHASE COLOR> is the color of the phase plot
  • <QUAD COLOR> is the color of the quad plot


INDICATOR,PLUGIN,HT,SINE,<INPUT>,<SINE NAME>,<LEAD NAME>,<SINE PLOT TYPE>,<LEAD PLOT TYPE><SINE COLOR>,<LEAD COLOR>

  • <INPUT> is an input type array
  • <SINE NAME> is the variable name given to the sine plot
  • <LEAD NAME> is the variable name given to the lead plot
  • <SINE PLOT TYPE> is the sine plot type
  • <LEAD PLOT TYPE> is the lead plot type
  • <SINE COLOR> is the color of the sine plot
  • <LEAD COLOR> is the color of the lead plot
