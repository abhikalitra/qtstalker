HT - Hilbert Transform


There are 7 different HT methods:

HT DCPERIOD
HT DCPHASE
HT PHASOR
HT QUAD
HT SINE
HT TRENDLINE
HT TRENDMODE



Script API

There are 8 different methods. They all have the same format.


Format:

INDICATOR,PLUGIN,HT,DCPERIOD,<NAME>,<INPUT>

INDICATOR,PLUGIN,HT,DCPHASE,<NAME>,<INPUT>

INDICATOR,PLUGIN,HT,TRENDLINE,<NAME>,<INPUT>

INDICATOR,PLUGIN,HT,TRENDMODE,<NAME>,<INPUT>

INDICATOR,PLUGIN,HT,PHASE,<NAME>,<INPUT>

INDICATOR,PLUGIN,HT,QUAD,<NAME>,<INPUT>

INDICATOR,PLUGIN,HT,SINE,<NAME>,<INPUT>

INDICATOR,PLUGIN,HT,LEAD,<NAME>,<INPUT>

  • <NAME> is the variable name given to the plot
  • <INPUT> is an input type array
