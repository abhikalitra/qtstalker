ROC - Rate of Change

There are 4 methods:
ROC, ROCP, ROCR, ROCR100

ROC

INDICATOR,PLUGIN,ROC,ROC,<NAME>,<INPUT>,<PERIOD>,<SMOOTHING PERIOD>,<SMOOTHING TYPE>

  • <NAME> is the variable name given to the plot
  • <INPUT> is an input type
  • <PERIOD> is an integer
  • <SMOOTHING PERIOD> is an integer
  • <SMOOTHING TYPE> MA type
