SETTINGS_LOAD


DESCRIPTION: Returns a setting from the main app settings. Setting is stored in KEY. This is only used in a few places where simple IPC is needed between the main app and scripts. Users should avoid this as it may be deprecated at any time.
SYNTAX: PLUGIN = SETTINGS_LOAD
KEY = <SETTINGS KEY>
RETURNS: '0' = Success
'ERROR' = Failure
KEY = SETTINGS_LOAD_DATA
EXAMPLE: PLUGIN=SETTINGS_LOAD,KEY=grid
