CORREL description.

CORREL

INDICATOR,PLUGIN,CORREL,<NAME>,<INPUT>,<INPUT 2>,<PERIOD>

  • <NAME> is the variable name given to the plot
  • <INPUT> is the input type
  • <INPUT 2> is the index composite to compare to ie. S&P 500
  • <PERIOD> an integer value used for the period
