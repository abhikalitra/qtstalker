LINEARREG - Linear Regression


There are 5 methods:

LINEARREG
LINEARREG ANGLE
LINEARREG INTERCEPT
LINEARREG SLOPE
TSF



Script API


Format:

INDICATOR,PLUGIN,LINEARREG,LINEARREG,<NAME>,<INPUT>,<PERIOD>

INDICATOR,PLUGIN,LINEARREG,ANGLE,<NAME>,<INPUT>,<PERIOD>

INDICATOR,PLUGIN,LINEARREG,INTERCEPT,<NAME>,<INPUT>,<PERIOD>

INDICATOR,PLUGIN,LINEARREG,SLOPE,<NAME>,<INPUT>,<PERIOD>

INDICATOR,PLUGIN,LINEARREG,TSF,<NAME>,<INPUT>,<PERIOD>

  • <NAME> is the variable name given to the plot
  • <INPUT> is an input type
  • <PERIOD> is an integer used for the period
