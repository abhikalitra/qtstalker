LINEARREG - Linear Regression

There are 5 methods:
LINEARREG, LINEARREG ANGLE, LINEARREG INTERCEPT, LINEARREG SLOPE, Time Series Forecast (TSF)

LINEARREG

INDICATOR,PLUGIN,LINEARREG,LINEARREG,<NAME>,<INPUT>,<PERIOD>

  • <NAME> is the variable name given to the plot
  • <INPUT> is an input type
  • <PERIOD> is an integer used for the period
