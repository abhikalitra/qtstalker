Futures Plugin

Description:

This plugin is used for managing futures data. User can edit bar data fields. There are 2 tabs in the stock dialog. The details tab allows the user to edit some fundamental data. The data tab allows the user to edit bar data.

Details Tab:

  • Symbol - This is the 'ticker' name. Uneditable.
  • Name - This is the contract name. Editable.
  • Type - This is the type of security ie stock, futures etc. Uneditable.
  • Futures Type - This is the exchange commodity code.
  • Futures Month - This is the contract expiry month code.

 


Data Tab:

Here is where you can edit the bar data. Just select a date and time and press the 'search' button and the data will display in the list. To edit a field, just select a field and change the values directly. To save your changes press the save button. To delete a record press the delete button.
