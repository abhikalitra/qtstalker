GROUP DATABASE SAVE_ALL


DESCRIPTION: Replaces all existing group NAME items with new ITEMS. ITEMS is a semicolon delimited string.
SYNTAX: PLUGIN = GROUP_DATABASE
METHOD = SAVE_ALL
NAME = <GROUP NAME>
ITEMS = <ITEMS> (semicolon delimited)
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=GROUP_DATABASE, METHOD=SAVE_ALL, NAME=group1, ITEMS=item1;item2
