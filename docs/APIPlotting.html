Script API - Plotting Indicators


PLOT,<NAME>,<LABEL>,<PLOT TYPE>

This function allows the indicator script to plot indicators.

  • <NAME> is the indicator previously created to plot.
  • <LABEL> is the string used to identify the plot on the chart.
  • <PLOT TYPE> is the type of plot desired. Available types are:
    • Dot - dotted line
    • Dash - dashed line
    • Histogram - histogram
    • Histogram Bar - histogram using vertical rectangles
    • Line - solid line
    • Horizontal - horizontal line across entire chart
On success returns "0", failure returns "1".
