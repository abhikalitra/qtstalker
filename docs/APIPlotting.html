Script API - Plotting Indicators


PLOT,<NAME>,<LABEL>,<PLOT TYPE>

This function allows the indicator script to plot indicators.

  • <NAME> is the indicator previously created to plot.
  • <LABEL> is the string used to identify the plot on the chart.
  • <PLOT TYPE> is the type of plot desired.


The available values of PLOT TYPE are:

  • Dot - draws a dotted line
  • Dash - draws a dashed line
  • Histogram - draws a histogram
  • HistogramBar - draws a histogram using vertical rectangles
  • Line - draws a solid line
  • Horizontal - draws a solid horizontal line


On success returns "0", failure returns "1".
