Indicators Panel

Description:

This panel is used to manage indicators. At the top of the panel are some buttons. The "Show active indicators" button will display all indicators that have been turned on. The "Show all indicators button" will list all indicators in the database. The "Search" button will search for a specific indicator(s) either an exact match or globbing using the % character for a partial match. If you want to flip on/off an individual indicator, just double click on the indicator in the list. The icon in the front of the list item displays the status of the indicator. A green check mark means 'on' and a red stop sign means 'off'.

Right clicking with the mouse on the list will display the context menu. It has the following functions.

New Indicator

Will display the New Indicator dialog. Allows the user to add a new indicator to the database. When added, the new indicator dialog will be displayed and turned on.

Edit Indicator

Allows the user to edit the parameters of the selected indicator.

Delete Indicator

Allows the user to delete selected indicator.
