Indicators Panel


Description:

The indicators tab allows you to flip on/off all the main chart indicators. This is useful if you have alot of plots that are cluttering up the screen. If you want to flip off all the indicators select the 'Disable All' checkbox at the bottom. If you want to flip on/off an individual indicator, just double click on the indicator in the list. The icon in the front of the list item displays the status of the indicator. A green check mark means 'on' and a red stop sign means 'off'.
