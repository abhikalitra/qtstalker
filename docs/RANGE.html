RANGE - Range Script Functions

These functions are for script use only. They are a collection of functions that act on a range of data points within an array.



Script API



Lowest value over <PERIOD>

INDICATOR,PLUGIN,RANGE,MIN,<NAME>,<INPUT>,<PERIOD>
  • <NAME> is the variable name given to the plot
  • <INPUT> is an input type
  • <PERIOD> is an integer
