QUOTE_DATABASE_EXCHANGE


DESCRIPTION: Returns a semicolon delimited string of all exchange names in the database. The string is stored in KEY.
SYNTAX: PLUGIN = QUOTE_DATABASE_EXCHANGE
RETURNS: '0' = Success
'ERROR' = Failure
KEY = QUOTE_DATABASE_EXCHANGE_EXCHANGES
EXAMPLE: PLUGIN=QUOTE_DATABASE_EXCHANGE
