INDICATOR_PLOT_COLOR_SET


DESCRIPTION: Sets INDEX to COLOR on curve NAME.
SYNTAX: PLUGIN = INDICATOR_PLOT_COLOR_SET
NAME = <VARIABLE NAME>
INDEX = <INTEGER>
COLOR = <COLOR>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=INDICATOR_PLOT_COLOR_SET,NAME=close,INDEX=100,COLOR=red
