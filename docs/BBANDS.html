BBANDS - Bollinger Bands

This is the standard Bollinger Bands indicator.


Script API


BBANDS format:

INDICATOR,PLUGIN,BBANDS,<INPUT>,<NAME UPPER>,<NAME MIDDLE>,<NAME LOWER>,<PERIOD>,<MA TYPE>,<UPPER DEVIATION>,<LOWER DEVIATION>,<UPPER PLOT TYPE>,<MIDDLE PLOT TYPE>,<LOWER PLOT TYPE>,<UPPER COLOR>,<MIDDLE COLOR>,<LOWER COLOR>

  • <INPUT> is either a previously created variable in a script, or one of these values
  • <NAME UPPER> is the variable name given to the upper band plot
  • <NAME MIDDLE> is the variable name given to the middle band plot
  • <NAME LOWER> is the variable name given to the lower band plot
  • <PERIOD> is an integer value for the period
  • <MA TYPE> is the moving average type to use
  • <UPPER DEVIATION> is a decimal value given for the upper band deviation
  • <LOWER DEVIATION> is a decimal value given for the lower band deviation
  • <UPPER PLOT TYPE> is the upper band plot type
  • <MIDDLE PLOT TYPE> is the middle band plot type
  • <LOWER PLOT TYPE> is the lower band plot type
  • <UPPER COLOR> is the color of the upper band
  • <MIDDLE COLOR> is the color of the middle band
  • <LOWER COLOR> is the color of the lower band
