BBANDS - Bollinger Bands


DESCRIPTION:    Bollinger Bands

SYNTAX:             PLUGIN = BBANDS
                           INPUT = <VARIABLE NAME>
                           NAME_UPPER = <VARIABLE NAME>
                           NAME_MIDDLE = <VARIABLE NAME>
                           NAME_LOWER = <VARIABLE NAME>
                           PERIOD = <INTEGER>
                           MA_TYPE = <MOVING AVERAGE TYPE>
                           DEV_UP = <DECIMAL>
                           DEV_DOWN = <DECIMAL>

RETURNS:           '0' = Success
                          'ERROR' = Failure

EXAMPLE:           PLUGIN=BBANDS,INPUT=close,NAME_UPPER=bbu,NAME_MIDDLE=bbm,NAME_LOWER=bbl,PERIOD=20,MA_TYPE=EMA,DEV_UP=2,DEV_DOWN=2
