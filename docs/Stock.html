Scripting API - Stock Quotes


QUOTE,Stock,SET_QUOTE,<EXCHANGE>,<SYMBOL>,<DATE FORMAT>,<DATE>,<OPEN>,<HIGH>,<LOW>,<CLOSE>,<VOLUME>

This function is used to save stock quotes into the database.
  • <EXCHANGE> is the exchange the quote originates from. ie. 'XNAS'
  • <SYMBOL> is the security symbol ie. GOOG
  • <DATE_FORMAT> is the date format.
  • <DATE> date value that conforms to the <DATE FORMAT> value
  • <OPEN> open value
  • <HIGH> high value
  • <LOW> low value
  • <CLOSE> close value
  • <VOLUME> volume value

For example. If we wanted to save a stock quote, the text would be:

"QUOTE,Stock,SET_QUOTE,XNAS,MSFT,yyyy-MM-dd HH:mm:ss,2010-03-09 13:24:00,25.50,27.25,23.75,26.25,123456789"

On success a '0' is returned. On failure, 'ERROR' is returned.


QUOTE,Stock,SAVE_QUOTES

This function will save stock quotes to the database after "SET_QUOTE" commands have been previously run in the script. If this command is not used after quote data has been "SET_QUOTE", then the quote data will be lost and NOT saved to the database once the script has completed.

Tip: This command should be used once after all quotes within a running script have been added. You will get a 1000% speed increase.

On success a '0' is returned. On failure, 'ERROR' is returned.




QUOTE,Stock,SET_NAME,<EXCHANGE>,<SYMBOL>,<NAME>

This function will set the company name of the symbol.
  • <EXCHANGE> is the exchange the symbol trades on. ie. 'XNAS'
  • <SYMBOL> is the symbol. ie. 'MSFT'
  • <NAME> is the company name to use. ie. 'Microsoft Corporation'

For example. Setting the company name for MSFT would be:

"QUOTE,Stock,SET_NAME,XNAS,MSFT,Microsoft Corporation"

 On success a '0' is returned. On failure, 'ERROR' is returned.




QUOTE,Stock,DELETE,<EXCHANGE>,<SYMBOL>

This function will permanently delete the symbol from the quotes database.
  • <EXCHANGE> is the exchange the symbol trades on. ie. 'XNAS'
  • <SYMBOL> is the symbol. ie. 'MSFT'

For example. Deleting MSFT from the database would be:

"QUOTE,Stock,DELETE,XNAS,MSFT"

 On success a '0' is returned. On failure, 'ERROR' is returned.



QUOTE,Stock,GET_QUOTES,<NAME>,<EXCHANGE>,<SYMBOL>,<BAR FIELD>,<BAR LENGTH>,<BARS>

This function will create an array of values using the bar field of the symbol requested. These values can be manipulated by the various INDICATOR group of script commands.

  • <NAME> is the variable given to the array to create
  • <EXCHANGE> is an exchange
  • <SYMBOL> is the quote symbol name
  • <BAR FIELD> is the bar field:
    • Open
    • High
    • Low
    • Close
    • Volume
    • OI
  • <BAR LENGTH> is the bar length:
    • 1 Minute
    • 5 Minute
    • 10 Minute
    • 15 Minute
    • 30 Minute
    • 60 Minute
    • Daily
    • Weekly
    • Monthly
  • <BARS> is the number of bars to create

For example, if we wanted to create an array of daily close values from the MSFT symbol, we would issue the command:

"QUOTE,Stock,GET_QUOTES,close_array,XNAS,MSFT,Close,Daily,275"

On success a '0' is returned. On failure, 'ERROR' is returned.




QUOTE,Stock,RENAME,<OLD EXCHANGE>,<OLD SYMBOL>,<NEW EXCHANGE>,<NEW SYMBOL>

This function will rename an existing symbol in the quotes database.
  • <OLD EXCHANGE> is the exchange of the symbol to be renamed
  • <OLD SYMBOL> is the stock symbol to be renamed
  • <NEW EXCHANGE> is the new exchange
  • <NEW SYMBOL> is the new symbol

On success a '0' is returned. On failure, 'ERROR' is returned.
