Qtstalker: Source code documentation

If this place-holder document is the front page of the Qt Help docs, then 'doxygen' has not yet been used to build the documentation.

Install Doxygen and ensure that the 'doxygen' executables are available on your $PATH

Optionally install Graphviz to generate better diagrams. Set HAVE_DOT=YES in the qtstalker/misc/doxygen/config.txt file.

Then do './build-api-docs.sh' and then do './configure' etc.
