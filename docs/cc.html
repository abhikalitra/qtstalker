CC Plugin

Description:

Creates a continuous adjusted chart for each futures type that Qtstalker can track. An adjusted chart uses the change in price from the previous bar instead of the actual price. This allows you to have a gapless long term futures chart.

Input Parameters:

  • Maximum Years - The maximum years back to include
  • Rebuild - If checked, the chart is rebuilt from scratch every time it is loaded. Unchecked, only missing bars since last loaded will be updated.
