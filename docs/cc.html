CC

Description:

Creates either a gapless/non-gapless chart for each futures type that Qtstalker can track. An gapless chart uses the change in price from the previous bar instead of the actual price. This allows you to have a gapless long term futures chart.

Input Parameters:

  • Futures Symbol - The futures symbol to use.
  • Gapless - If checked, a gapless chart is created.
