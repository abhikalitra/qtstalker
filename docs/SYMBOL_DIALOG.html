SYMBOL_DIALOG


DESCRIPTION: Displays the symbol selection dialog.
SYNTAX: PLUGIN = SYMBOL_DIALOG
FLAG = <BOOLEAN>
  1. 0 - return symbols in KEY = SYMBOL_DIALOG_SYMBOLS
  2. 1 - return exchange/symbol search pattern in KEY = SYMBOL_DIALOG_EXCHANGE, KEY = SYMBOL_DIALOG_SYMBOL
RETURNS: '0' = Success
'ERROR' = Failure
KEY = SYMBOL_DIALOG_SYMBOLS (semicolon delimited)
KEY = SYMBOL_DIALOG_EXCHANGE
KEY = SYMBOL_DIALOG_SYMBOL
EXAMPLE: PLUGIN=SYMBOL_DIALOG,FLAG=0
