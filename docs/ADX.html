ADX - Average Directional Index


The qtstalker version of ADX allows the user to plot any number of the 4 ADX components. The +DI, -DI, ADX and the ADXR.




Script API


ADX format:

INDICATOR,PLUGIN,ADX,ADX,<NAME>,<PERIOD>
  • <NAME> is the variable name given to the ADX plot
  • <PERIOD> is the ADX period

ADXR format:

INDICATOR,PLUGIN,ADX,ADXR,<NAME>,<PERIOD>

  • <NAME> is the variable name given to the ADXR plot
  • <PERIOD> is the ADXR period

DX format:

INDICATOR,PLUGIN,ADX,DX,<NAME>,<PERIOD>

  • <NAME> is the variable name given to the DX plot
  • <PERIOD> is the DX period

-DI format:

INDICATOR,PLUGIN,ADX,-DI,<NAME>,<PERIOD>

  • <NAME> is the variable name given to the -DI plot
  • <PERIOD> is the -DI period

+DI format:

INDICATOR,PLUGIN,ADX,+DI,<NAME>,<PERIOD>

  • <NAME> is the variable name given to the +DI plot
  • <PERIOD> is the +DI period

-DM format:

INDICATOR,PLUGIN,ADX,-DM,<NAME>,<PERIOD>

  • <NAME> is the variable name given to the -DM plot
  • <PERIOD> is the -DM period

+DM format:

INDICATOR,PLUGIN,ADX,+DM,<NAME>,<PERIOD>

  • <NAME> is the variable name given to the +DM plot
  • <PERIOD> is the +DM period
