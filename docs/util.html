Utilities - UTIL


Parameters:

  • Method - the function type to use
    • ACCUM - accumulation
    • TypPrice - WIlder's typical price
    • Normal - normalize data
    • ADD - addition
    • DIV - division
    • MUL - multiplication
    • SUB - subtraction
    • COMP - compare one number with another using relational operators
    • COUNTER
    • REF - return an array of bars offset backwards

Description:

A collection of functions for use in creating custom indicators with the CUS indicator.

Formula Reference:

ACCUM Syntax
 UTIL(Method, ARRAY INPUT)
Function
 ARRAY INPUT can be either an array of bars or a formula variable array of data.
Example
 UTIL(ACCUM, Close)

TypPrice
 UTIL(Method)
Function
Example
 UTIL(TypPrice)

Normal Syntax
 UTIL(Method, ARRAY INPUT)
Function
 ARRAY INPUT can be either an array of bars or a formula variable array of data.
Example
 UTIL(Normal, Close)

ADD Syntax
 UTIL(Method, ARRAY INPUT, DOUBLE)
UTIL(Method, ARRAY INPUT, ARRAY INPUT2)
Function
 Adds the values of the ARRAY INPUT and ARRAY INPUT2/DOUBLE value. ARRAY INPUT and ARRAY INPUT2 can be either an array of bars or a formula variable array of data. The DOUBLE is a number.
Example
 UTIL(ADD, Close, Open) or UTIL(ADD, Close, 45.5)

UTIL(Method, ARRAY INPUT, ARRAY INPUT2)
Function

Example

COMP Syntax

UTIL(Method, ARRAY INPUT, ARRAY INPUT2, OPERATOR)
Function

Function

REF Syntax
 UTIL(Method, ARRAY INPUT, PERIOD)
Function
 ARRAY INPUT can be either an array of bars or a formula variable array of data. PERIOD is the number of bars to offset backwards.
Example
 UTIL(REF, Close, 0) for no offset, UTIL(REF, Close, 10) to offset 10 bars backwards
