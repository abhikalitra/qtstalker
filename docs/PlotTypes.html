Plot Types

The available plot types are:

  • Bars - standard OHLC bars
  • Candle - Japanese candlesticks
  • Dot - a line of dots
  • Histogram - a histogram plot centered on a zero y axis
  • HistogramBar - a histogram / bar plot hybrid centered on a zero y axis
  • HistogramHorizontal - horizontal histogram on left side of chart. only used for the VBP indicator
  • Line - a solid line
