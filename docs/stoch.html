Stochastic Oscillator - STOCH


Parameters:

  • Method - Standard or Adaptive stochastic method.
  • Input - The input bar field to use.
  • %D Color - %D color
  • %D Line Type - %D line type
  • %D Label - The text to identify the %D plot
  • %D Smoothing - %D smoothing period
  • %D Smoothing Type - %D smoothing type
  • %K Color - %K color
  • %K Line Type - %K line type
  • %K Label - The text to identify the %K plot
  • %K Smoothing - %K smoothing period
  • %K Smoothing Type - %K smoothing type
  • Buy Line - Buy signal threshold
  • Sell Line - Sell signal threshold
  • Period - STOCH period
  • Min Lookback - The minimum lookback period for the Adaptive method only.
  • Max Lookback - The maximum lookback period for the Adaptive method only.

Description:

Standard
This is the standard stochastic formula.

Singular
This is the stochastic formula applied to a single input array of data. The high/low values are found within the input array. The 'close' is the current value in the array. This allows the user to apply the standard formula to any type of data.

Adaptive

Based on Tushar Chande's adaptave approach:

"The strategy is to change the lookback length of the indicator by connecting it to market volatility. We
would like the indicator to have a long lookback period when prices are in a trading range (volatility is  low). Conversely, we want a short lookback length when prices are moving rapidly. I feel that changing the lookback length for calculating indicators is more responsive than smoothing them with adaptive moving averages. The actual definition of long and short lookback lengths depends on your trading horizon."

The volitility component in the indicator is the standard deviation or the close (configurable to any number of days --the default is set to 14). Also configurable is the smoothing component for both %K and %D.

Formula Reference:

Syntax
 STOCH(Method, %K Smoothing Type, Period, %K Smoothing) *this is the standard format

STOCH(Method, Input, %K Smoothing Type, Period, %K Smoothing)  *this is the singular format

STOCH(Method, Input, %K Smoothing Type, Period, %K Smoothing, Min Lookback, Max Lookback)  *this is the adaptive format
Function
 Input can be either an array of bars or a formula variable array of data. Method is either Standard, Singular or Adaptive.
Example
 STOCH(Standard, EMA, 14, 3)
STOCH(Singular, Close, EMA, 14, 3)
STOCH(Adaptive, Close, EMA, 14, 3, 5, 20)
