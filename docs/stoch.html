Stochastic Oscillator - STOCH


Parameters:

  • Method - Standard or Adaptive stochastic method.
  • Input - The input bar field to use.
  • %D Color - %D color
  • %D Line Type - %D line type
  • %D Label - The text to identify the %D plot
  • %D Smoothing - %D smoothing period
  • %D Smoothing Type - %D smoothing type
  • %K Color - %K color
  • %K Line Type - %K line type
  • %K Label - The text to identify the %K plot
  • %K Smoothing - %K smoothing period
  • %K Smoothing Type - %K smoothing type
  • Buy Line - Buy signal threshold
  • Sell Line - Sell signal threshold
  • Period - STOCH period
  • Min Lookback - The minimum lookback period for the Adaptive method only.
  • Max Lookback - The maximum lookback period for the Adaptive method only.

Description:


Formula Reference:

Syntax
 STOCH(Input, Method, %K Smoothing Type, %D Smoothing Type, Period, %K Smoothing, %D Smoothing, Min Lookback, Max Lookback)
Function
 Input can be either an array of bars or a formula variable array of data. Method is either Standard or Adaptive.
Example
 STOCH(Close, Standard, EMA, EMA, 14, 3, 3, 0, 0) or
STOCH(Close, Adaptive, EMA, EMA, 14, 3, 3, 5, 20)
