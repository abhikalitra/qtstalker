Adaptive Stochastic - AdaptSTOCH

 

Parameters:

  • Period - Adaptive period.
  • Min Lookback - Min lookback period.
  • Max Lookback - Max lookback period.
  • %K Color - Plot color
  • %K Line Type - The type of plot to draw
  • %K Label - The text that identifies the plot
  • %K Smoothing Type - The type of MA to use for smoothing
  • %K Smoothing Period - The MA period used for smoothing
  • %D Color - Plot color
  • %D Line Type - The type of plot to draw
  • %D Label - The text that identifies the plot
  • %D Smoothing Type - The type of MA to use for smoothing
  • %D Smoothing Period - The MA period used for smoothing
  • Buy Zone Color -
  • Sell Zone Color -
  • Buy Zone - The buy zone value
  • Sell Zone - The sell zone value

 AdaptSTOCH(ARRAY INPUT, %K Smoothing Type, %D Smoothing Type, Period, %K Smoothing, %D Smoothing, Min Lookback, Max Lookback)
