REF - Reference

This function is only for use in scripts.

Creates an array that is <INPUT> - <PERIOD> offset. So, if you wanted to create a dataset of yesterday's close values the string would be "INDICATOR,PLUGIN,REF,yclose,Close,1" If you wanted today's Close then <PERIOD> would be '0'


Script API

INDICATOR,PLUGIN,REF,<NAME>,<INPUT>,<PERIOD>

  • <NAME> is the variable name given to the plot
  • <INPUT> is an input type
  • <PERIOD> is an integer
