Volatility - VOLA

 

Parameters:

  • Color - plot color
  • Line Type - The type of plot to draw
  • Label - The text that identifies the plot
  • Period - The VIDYA period
  • Input - The input bars. Used only for the SD method
  • Method
    • CV - Chaikin's Volatility
    • SD - Standard Deviation
    • VOLR - Volatility Ratio

Description:


Formula Reference:

Syntax
 VOLA(Method, Period, Input)
Function
 Input is used only for the SD method. It can be either an array of bars or a formula variable array of data.
Example
 VOLA(CV, 14) or VOLA(SD, 14, Close)
