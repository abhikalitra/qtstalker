Qtstalker: Documentation

Local documentation can be viewed with a web browser, by opening /usr/local/share/doc/qtstalker/html/index.html

Source code documentation can be viewed with a web browser, by opening path-to-your-source/qtstalker/docs/api/index.html (if documentation has not yet been built using Doxygen, then see the file api.html in that directory).

Documentation can also be viewed via the Qt Assistant application, by selecting from the Qtstalker "Help" menu.
