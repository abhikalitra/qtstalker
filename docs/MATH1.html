MATH1 - Script Utilities

These functions are only for use in scripts. They are a collection of math functions that work on variable arrays.



Script API




Creates a normalized array from <INPUT> (scales values from 0 - 100)

INDICATOR,PLUGIN,NORMALIZE,<NAME>,<INPUT>
  • <NAME> is the variable name given to the array
  • <INPUT> is an input type


Adds <INPUT> and <INPUT 2> together.

INDICATOR,PLUGIN,ADD,<NAME>,<INPUT>,<INPUT 2>
  • <NAME> is the variable name given to the array
  • <INPUT> is an input type
  • <INPUT 2> is an input type


Divides <INPUT> by <INPUT 2>

INDICATOR,PLUGIN,DIV,<NAME>,<INPUT>,<INPUT 2>
  • <NAME> is the variable name given to the array
  • <INPUT> is an input type
  • <INPUT 2> is an input type


Multiplies <INPUT> by <INPUT 2>

INDICATOR,PLUGIN,MULT,<NAME>,<INPUT>,<INPUT 2>
  • <NAME> is the variable name given to the array
  • <INPUT> is an input type
  • <INPUT 2> is an input type


Subtracts <INPUT 2> from <INPUT>
INDICATOR,PLUGIN,SUB,<NAME>,<INPUT>,<INPUT 2>
  • <NAME> is the variable name given to the array
  • <INPUT> is an input type
  • <INPUT 2> is an input type


Standard Deviation

INDICATOR,PLUGIN,STDDEV,<NAME>,<INPUT>,<PERIOD>,<DEVIATION>
  • <NAME> is the variable name given to the plot
  • <INPUT> is an input type
  • <PERIOD> an integer value
  • <DEVIATION> a decimal value
