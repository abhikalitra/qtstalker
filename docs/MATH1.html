MATH1 - Script Utilities

These functions are only for use in scripts. They are a collection of math functions that work on variable arrays.



Script API



Standard Ceiling function.

INDICATOR,PLUGIN,MATH1,CEIL,<NAME>,<INPUT>
  • <NAME> is the variable name given to the array
  • <INPUT> is an input type
