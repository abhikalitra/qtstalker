SHIFT


DESCRIPTION: Shifts the all values for PERIOD. If PERIOD < 0 then values are shifted back in time. If PERIOD is > 0, then values are shifted forward in time.
SYNTAX: PLUGIN = SHIFT
NAME = <VARIABLE NAME>
INPUT = <VARIABLE NAME>
PERIOD = <INTEGER>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=SHIFT, NAME=shift, INPUT=close, PERIOD=10
