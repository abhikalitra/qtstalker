Quotes

Description

This is the quote update center. You use the various quote plugins to create and update the data you want and import them into Qtstalker. Each plugin will use different input data parameters. Once all the input parameters are satisfied, pressing the "Update" button will start the update. If you want to interrupt an update, press the "Cancel Update" button. You may change the settings by selecting the setting with the mouse. Depending on the setting type, a dialog will be popup and allow you to select and enter data.

The available plugins are:

  • CBOT
  • CME
  • CMEHistory
  • COT
  • Index
  • NYBOT
  • Qtstalker
  • Spread
  • Yahoo
  • YahooQuote

The button functions are as follows:

Ok

This is disabled as it is not needed.

Cancel

Exits the dialog.

Update

Starts the update for all charts that apply for the selected plugin.

Cancel Update

Interrupts any pending updates. Only available when an update is running.

New Chart

This button is available only if it required by the plugin. eg. Yahoo uses this function as a way to input new symbols to download. Pressing this will bring up a dialog to enter parameters needed to create new charts for the selected plugin.
