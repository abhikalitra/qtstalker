FILE_DIALOG


DESCRIPTION: Displays a file selection dialog. Returns the selected file(s) as a semicolon delimited string as KEY.
SYNTAX: PLUGIN = FILE_DIALOG
MODE = <BOOLEAN>
  1. 0 - single selection
  2. 1 - multiple selection
FORMAT = <BOOLEAN>
  1. 0 - returns complete file path e.g. /usr/local/share/qtstalker/somefile.pl
  2. 1 - returns only the filename e.g. somefile.pl
DIRECTORY = <INITIAL DIRECTORY PATH>
RETURNS: '0' = Success
'ERROR' = Failure
KEY=FILE_DIALOG_FILE
EXAMPLE: PLUGIN=FILE_DIALOG,MODE=1,FORMAT=0,DIRECTORY=/usr/local/share/qtstalker
