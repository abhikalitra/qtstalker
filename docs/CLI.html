Command Line Interface (CLI)

The available command line options and switches are listed here.

-session <ID>

  • <ID> is an integer

Allows multiple instances of qtstalker to run by creating a unique session id. The deafult is session 0 if not used. So for example if I wanted to run 2 concurrent instances of qtstalker each with its own settings, I would issue 2 commands:

  1. 'qtstalker &' (uses the deafult session 0)
  2. 'qtstalker -session 1 &' (creates a second instance called session 1)

-asset <EXCHANGE>:<SYMBOL>

  • <EXCHANGE> is the exchange the asset trades on. e.g. 'XNYS'
  • <SYMBOL> is the asset symbol e.g. 'GLD'

Will display a chart of the <EXCHANGE>:<SYMBOL> upon starting. Useful for interacting with other applications and scripts. For example:

  1. 'qtstalker -asset XNYS:GLD &'   would display the GLD chart upon starting.
