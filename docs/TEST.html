TEST


DESCRIPTION: Conducts a back test of a symbol depending on the parameters set. This plugin operates a little different from the others. Each METHOD type has it's own sub commands that only apply to that particular method.
SYNTAX: PLUGIN = TEST
METHOD =
  1. SET_RULE
    1. TYPE =
      1. <ENTER_LONG>
      2. <EXIT_LONG>
      3. <ENTER_SHORT>
      4. <EXIT_SHORT>
    2. NAME = <VARIABLE NAME>
    3. OP = <OPERATOR>
    4. NAME2 = <VARIABLE NAME>
  2. TEST
  3. SET_VOLUME
    1. VOLUME = <INTEGER>
  4. SET_ENTER_COMM
    1. COMM = <DECIMAL>
  5. SET_EXIT_COMM
    1. COMM = <DECIMAL>
  6. SET_EQUITY
    1. EQUITY = <DECIMAL>
  7. SET_PRICES
    1. BUY = <VARIABLE NAME>
    2. SELL = <VARIABLE NAME>
    3. CLOSE = <VARIABLE NAME>
  8. RESET
  9. SAVE
    1. NAME = <TEXT>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=TEST
