TEST


DESCRIPTION: Conducts a back test of a symbol depending on the parameters set. The results are saved to the database. If using optimization, then the best result (highest equity) will be saved.
SYNTAX: PLUGIN = TEST
NAME = <TEST NAME> - a unique name to identify the test
SYMBOL = <EXCHANGE:SYMBOL> - the security symbol
ENTER_LONG_NAME = <VARIABLE NAME> - *optional if not testing long trades
ENTER_LONG_OP = <OPERATOR> - *optional if not testing long trades
ENTER_LONG_NAME2 = <VARIABLE NAME> - *optional if not testing long trades
EXIT_LONG_NAME = <VARIABLE NAME> - *optional if not testing long trades
EXIT_LONG_OP = <OPERATOR> - *optional if not testing long trades
EXIT_LONG_NAME2 = <VARIABLE NAME> - *optional if not testing long trades
ENTER_SHORT_NAME = <VARIABLE NAME> - *optional if not testing short trades
ENTER_SHORT_OP = <OPERATOR> - *optional if not testing short trades
ENTER_SHORT_NAME2 = <VARIABLE NAME> - *optional if not testing short trades
EXIT_SHORT_NAME = <VARIABLE NAME> - *optional if not testing short trades
EXIT_SHORT_OP = <OPERATOR> - *optional if not testing short trades
EXIT_SHORT_NAME2 = <VARIABLE NAME> - *optional if not testing short trades
DATE_NAME = <VARIABLE NAME> - dates used for trades
BUY_NAME = <VARIABLE NAME> - prices used for entering trades
SELL_NAME = <VARIABLE NAME> - prices used for exiting trades
CLOSE_NAME = <VARIABLE NAME> - prices used for EOD open trades
VOLUME = <DECIMAL> *optional (0.02 default 2%) - volume percentage of current equity to calculate volume. If set to zero, a volume of 1 will be used. 0 - 1 range.
ENTER_COMM = <DECIMAL> *optional (10 default) - entry commission
EXIT_COMM = <DECIMAL> *optional (10 default) - exit commission
EQUITY = <DECIMAL> *optional (1000 default) - starting equity
DELAY = <INTEGER> *optional (0 default) - delay trade entry/exit 1 bar
MAX_LOSS_STOP = <DECIMAL> *optional (0 default) exit trade if loss => the entry value * MAX_LOSS_STOP. 0 - 1 range.
TRAILING_STOP = <DECIMAL> *optional (0 default) 0 - 1.0 range.
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=TEST, NAME=test1, SYMBOL=XNYS:GOOG, ENTER_LONG_NAME=close, ENTER_LONG_OP=GT, ENTER_LONG_NAME2=emaHigh, EXIT_LONG_NAME=close, EXIT_LONG_OP=LT, EXIT_LONG_NAME2=emaLow, DATE_NAME=date, BUY_NAME=open, SELL_NAME=close, CLOSE_NAME=close, DELAY=1, MAX_LOSS_STOP=0.06, TRAILING_STOP=0.06
