Spread Plugin

Description:

Create and update spread charts. A spread chart is the value of the first symbol minus the value of the second symbol. Usually used by futures traders, but can be used with any stock or futures chart.

Input Parameters:

  • First Symbol - The 'buy' half of the spread.
  • Second Symbol - The 'sell' half of the spread.
  • Method - Options for the spread method. Subtract, Divide are the current options.
  • Rebuild - Checked will rebuild the complete chart. Unchecked will update missing bar from last bar.

The subtract method will calculate the first symbol minus the second symbol. The chart will reflect the "spread" between the two charts as a difference positive or negative.

The divide method will calculate the first symbol divided by the second symbol. The chart will reflect the ratio between the two symbols.
