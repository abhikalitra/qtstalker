Script API - Symbols


SYMBOL,CURRENT

This function returns a csv string of the currently loaded exchange and symbol to the script.

An example would return "NASDAQ,MSFT"

On success the exchange symbol pair is returned. On failure, the string 'ERROR' is returned.



SYMBOL,SEARCH,<EXCHANGE>,<SEARCH STRING>

This function allow scripts to return a csv string of exchange,symbol pairs in the quote database. If both fields are left blank, then all symbols in the database will be retrurned.
  • <EXCHANGE> Specifies a particular exchange to search on. If left blank, exchange field will be ignored and all symbols from all exchanges that match <SEARCH STRING> will be returned.
  • <SEARCH STRING> Is the search string pattern. The % is used to match any length. ie %IBM% would return any symbols containing IBM. The '_' character is used to match on a single character. If left blank, then all symbols that match the <EXCHANGE> field will be returned.

An example would be "SYMBOL,SEARCH,NASDAQ,%MSFT%"
The returned string would be "NASDAQ,MSFT,NASDAQ,MSFTX,NASDAQ,MSFTY"

On success a csv string of exchange,symbol pairs is returned. On failure, the string "ERROR" is returned.
