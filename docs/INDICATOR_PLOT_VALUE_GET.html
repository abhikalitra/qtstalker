INDICATOR_PLOT_VALUE_GET


DESCRIPTION: Returns the value of NAME at INDEX and stores it in KEY.
SYNTAX: PLUGIN = INDICATOR_PLOT_VALUE_GET
NAME = <VARIABLE NAME>
INDEX = <INTEGER>
RETURNS: '0' = Success
'ERROR' = Failure
KEY = INDICATOR_PLOT_VALUE_GET_VALUE
EXAMPLE: PLUGIN=INDICATOR_PLOT_VALUE_GET,NAME=close,INDEX=100
