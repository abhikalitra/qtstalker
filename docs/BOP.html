BOP - Balance of Power

Description



Script API



INDICATOR,PLUGIN,BOP,<NAME>,<SMOOTHING PERIOD>,<SMOOTHING TYPE>,<PLOT TYPE>,<COLOR>
  • <NAME> is the variable name given to the array
  • <SMOOTHING PERIOD> an integer value used for the smoothing period
  • <SMOOTHING TYPE> the MA type used for smoothing
  • <PLOT TYPE> is the plot type
  • <COLOR> is the color of the plot
