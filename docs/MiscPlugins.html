Misc. Plugins

QtStalker has a plugin system that allows for extendable feature set to expand over time. These plugins are accessable using the Plugins page in the side panel of QtStalker. To bring up a dialog for the plugin, either double click with the mouse or select the plugin in the list and press the configure button. Anyone may write a plugin provided they have the C++ skills to.
