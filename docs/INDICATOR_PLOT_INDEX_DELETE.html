INDICATOR_PLOT_INDEX_DELETE


DESCRIPTION: Deletes a single value from NAME at position INDEX.
SYNTAX: PLUGIN = INDICATOR_PLOT_INDEX_DELETE
NAME = <VARIABLE NAME>
INDEX = <INTEGER>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=INDICATOR_PLOT_INDEX_DELETE,NAME=close,INDEX=100
