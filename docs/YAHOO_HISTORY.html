YAHOO_HISTORY


DESCRIPTION: Downloads CSV quotes from Yahoo Finance and stores them in a CSV_FILE text file. DATE_START and DATE_END are the date ranges to download. The symbols to download are read from SYMBOL_FILE user managed file. The ADJUSTED setting selects using split adjusted data rather than the actual trading values.
SYNTAX: COMMAND = YAHOO_HISTORY
STEP = <VARIABLE NAME>
DATE_START = <yyyy-MM-dd HH:mm:ss>
DATE_END = <yyyy-MM-dd HH:mm:ss>
SYMBOL_FILE = <FILE_PATH>
CSV_FILE = <FILE_PATH>
ADJUSTED = <BOOLEAN>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: COMMAND=YAHOO_HISTORY;
STEP=yh;
DATE_START=2000-01-01 09:30:00;
DATE_END=2011-08-22 16:30:00;
SYMBOL_FILE=/home/user/qtstalker_stuff/yahoo.symbols
CSV_FILE=/home/user/qtstalker_stuff/yahoo.csv
ADJUSTED=false
