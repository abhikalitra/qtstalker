INPUT_DIALOG


DESCRIPTION: Displays a dialog that allows various input types such as integers, decimals, text etc. Up to 10 different inputs can be defined at once. The values returned are located in KEY, one for each INPUT.
SYNTAX: PLUGIN = INPUT_DIALOG
INPUT1_LABEL = <TEXT> - the input label text that appears to the left of the input.
INPUT1_TYPE =
  • INTEGER
  • DOUBLE
  • TEXT
  • DATE
INPUT1_VALUE = <INITIAL VALUE> - the default value to display
...
...
...
INPUT10_LABEL = <TEXT>
INPUT10_TYPE = <TYPE>
INPUT10_VALUE = <INITIAL VALUE>
RETURNS: '0' = Success
'ERROR' = Failure
KEY = INPUT_DIALOG_INPUT1_DATA
KEY = INPUT_DIALOG_INPUT2_DATA
...
...
...
KEY = INPUT_DIALOG_INPUT10_DATA
EXAMPLE: PLUGIN=INPUT_DIALOG,INPUT1_LABEL=Enter minimum period,INPUT1_TYPE=INTEGER,INPUT1_VALUE=2
