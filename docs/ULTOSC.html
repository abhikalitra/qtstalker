ULTOSC - Ultimate Oscillator

Description



Script API

INDICATOR,PLUGIN,ULTOSC,<NAME>,<PERIOD SHORT>,<PERIOD MEDIUM>,<PERIOD LONG>

  • <NAME> is the variable name given to the array
  • <PERIOD SHORT> is an integer ( 7 is suggested)
  • <PERIOD MEDIUM> is an integer ( 14 is suggested)
  • <PERIOD LONG> is an integer ( 28 is suggested)
