SETTINGS_SAVE


DESCRIPTION: Saves KEY / DATA pair to the main app settings file. This is only used in a few places where simple IPC is needed between the main app and scripts. Users should avoid this as it may be deprecated at any time.
SYNTAX: PLUGIN = SETTINGS_SAVE
KEY = <SETTINGS KEY>
DATA = <SETTINGS DATA>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=SETTINGS_SAVE,KEY=grid,DATA=1
