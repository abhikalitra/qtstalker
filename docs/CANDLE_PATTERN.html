CANDLE_PATTERN


DESCRIPTION: Various traditional Japanese candlestick patterns. The PENETRATION value is the percentage of candle penetration used for certain METHOD types.

NAME = <VARIABLE NAME>

  3. 3INSIDE

  16. DOJI
  17. DOJISTAR
  18. DRAGONFLYDOJI

  21. EVENINGSTAR (uses PENETRATION)
  22. GAPSIDESIDEWHITE
  23. GRAVESTONEDOJI
  24. HAMMER
  25. HANGINGMAN
  26. HARAMI

  59. UNIQUE3RIVER
  60. UPSIDEGAP2CROWS
  61. XSIDEGAP3METHODS

RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=CANDLE_METHOD,NAME=harami,METHOD=HARAMI
PLUGIN=CANDLE_METHOD,NAME=dcc,METHOD=DARKCLOUDCOVER,PENETRATION=0
