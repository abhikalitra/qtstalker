CMO - Chande Momentum Oscillator


CMO description.



Script API


Format:

INDICATOR,PLUGIN,CMO,<NAME>,<INPUT>,<PERIOD>,<PLOT TYPE>,<COLOR>
  • <NAME> is the variable name given to the plot
  • <INPUT> is the input type
  • <PERIOD> an integer value used for the period
  • <PLOT TYPE> is the plot type
  • <COLOR> is the color of the plot
