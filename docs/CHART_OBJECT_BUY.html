CHART_OBJECT_BUY


DESCRIPTION: Displays an upward pointing arrow on an indicator. Used to indicate a buy point.

SYNTAX: PLUGIN = CHART_OBJECT_BUY
NAME = <VARIABLE NAME>
INDICATOR = <INDICATOR NAME>
EXCHANGE = <EXCHANGE>
SYMBOL = <SYMBOL>
DATE = <yyyy-MM-dd HH:mm:ss>
PRICE = <DECIMAL>
COLOR = <COLOR>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=CHART_OBJECT_BUY,NAME=buy,INDICATOR=Bars,EXCHANGE=XNYS,SYMBOL=GOOG,DATE=2011-02-08 16:00:00,PRICE=450.50,COLOR=green
