Custom - CUS



Description:

Allows the user to create custom indicators to plot. The user builds an indicator in a step by step manner. Using the above pic as an example to build a copy of the built in MACD indicator manually. The first function uses the OSC function to create a MACD. We check the Plot column here if we want this particular function to appear on the plot. Step 2 then creates the trigger plot by applying an EMA using step 1 as the input. We also want to plot this. Step 3 then builds a histogram of  the MACD and trigger plots by subtracting step 2 from step 1 using the SUB function. We check this to plot as well.

By using all the functions available, we are able to build almost any kind of indicator quickly and simply without having to learn a new scripting language. Complicated intensive indicators would best be left to creating a plugin instead of long series of functions for speed and memory considerations.

Function Reference:

  • data1 - This is a data array of values. It can either be an array of bar fields like (Open, High, Low, Close, Volume or Open Interest), or it can refer to a previous function in the formula. ie 1 refering to line 1 in the formula.
  • data2 - This is a data array of values. It allows 3 different values. It behaves exactly like data1 as well as being able to use a constant value. ie #10.375. To specify a value constant use the # sign to distinguish between a constant value and a formula line number.
  • color - The color assigned to the plot.
  • line type - The line type in which to plot.
  • label - The text used to identify the plot in the data window and plot.
  • matype - The type of moving average to use.
  • operator - The comparison operator to use ie. <=

NAME Accumulation Distribution
SYNTAX
 AD,color,line type,label
FUNCTION
 Calculates the predefined Accumulation/Distribution indicator. See AD indicator for more info.
EXAMPLE
 "ADD,red,Line,AD"

NAME Addition
SYNTAX
 ADD,data1,data2,color,line type,label
FUNCTION
 The sum of data1 and data2.
EXAMPLE
 "ADD,Close,Open,red,Line,test" adds the close and open and plots a 'red' line type with label 'test'
"ADD,2,Close,yellow,Dash,test" adds the result of line 2 of the formula to the close, and plots a yellow dashed line with 'test' as the label.
"ADD,Close,#10.375,green,Histogram,test" adds the close and a constant value of 10.375, and plots a 'green' 'histogram' with label 'test'.

NAME Compare
SYNTAX
 COMP,data1,operator,data2,color,line type,label
FUNCTION
 Compares data1 to data2 and returns a TRUE or FALSE.
EXAMPLE
 "COMP,Close,>,Open,green,Line,Buy" if the close is greater than the open returns TRUE else returns FALSE.
"COMP,Close,<,#10.375,red,Line,Sell" if the close is less than the constant 10.375 returns TRUE else returns FALSE.
