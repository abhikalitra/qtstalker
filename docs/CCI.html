CCI - Commodity Channel Index

CCI- is the standard version.



Script API

Format:

INDICATOR,PLUGIN,CCI,<NAME>,<PERIOD>,<SMOOTHING PERIOD>,<SMOOTHING TYPE>,<PLOT TYPE>,<COLOR>
  • <NAME> is the variable name given to the ADOSC plot
  • <PERIOD> is an integer for the CCI period value
  • <SMOOTHING PERIOD> is an integer used for the smoothing period
  • <SMOOTHING TYPE> is the MA type used for smoothing
  • <PLOT TYPE> is the plot type
  • <COLOR> is the color of the plot
