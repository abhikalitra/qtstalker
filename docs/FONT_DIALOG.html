FONT_DIALOG


DESCRIPTION: Displays a font selection dialog. Font selected in stored in KEY as a semicolon delimited string.
SYNTAX: PLUGIN = FONT_DIALOG
FONT= <FONT> (semicolon delimited string)
RETURNS: '0' = Success
'ERROR' = Failure
KEY = FONT_DIALOG_FONT
EXAMPLE: PLUGIN=FONT_DIALOG
