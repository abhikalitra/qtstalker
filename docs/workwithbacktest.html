Backtest Panel

Description:

The backtest tab allows you to open, edit, delete, backtesting rules. A backtesting rule is a collection of indicators and stops used to perform trades based on real chart data. The results are then recorded, listed and graphed for you. The purpose is to allow you to test "what if" ideas without risking real money.



The context menu functions are as follows:

Open Backtest Rule

This opens a backtest rule for editing or testing and displays the Back Tester dialog.

New Backtest Rule

This creates a new backtest rule.

Delete Backtest Rule

Permanently delete the backtest rule selected.

Rename Backtest Rule

Brings up a dialog that allows you to rename the selected backtest rule.

Copy Backtest Rule

Allows the user to create a copy of an existing rule.
