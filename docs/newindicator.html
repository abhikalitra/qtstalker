New Indicator

Description:

Create a new indicator for display.

Selecting New Indicator will bring up a dialog that asks you to choose from the list of installed indicator plugins. Only indicators that have been installed as plugins will be available. The "Name" field requires a unique name for the new indicator. The "Plot Type" field gives you the choice of where you want the indicator to be placed. The three areas available are:

  • Main - Indicator will be added to the main plot. Not all indicators will plot nicely due to scaling differences. ie. VOL
  • Tabbed - Indicator will be added to the indicator tab area at the bottom of the app.
  • Stacked - Indicator will be stacked above the indicator tab area.



After completing the above dialog, the edit indicator dialog will be displayed. All of the parameters that apply to the indicator can be edited here.
