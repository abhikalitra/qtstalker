Basic Installation

Requirements

  1. A Linux or FreeBSD compatible operating system.
  2. At least the Qt 3.* development library.

Installing from source

  1. Unpack the Qtstalker source.
  2. ./configure (this will compile the DB lib and create the Makefile)
  3. make
  4. make install (as root)
    • the executable 'qtstalker' will be copied to the /usr/bin directory
    • the plugins will be copied to /usr/lib/qtstalker directory
    • the docs will be copied to /usr/share/doc/qtstalker directory (toc.html for the table of contents)
  5. Thats it.
