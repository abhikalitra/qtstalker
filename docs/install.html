Basic Installation

Requirements

  1. A Linux compatible operating system.
  2. At least Qt 3.* development library.

Installing from source

  1. Unpack the Qtstalker source.
  2. cd into the qtstalker* directory.
  3. Run the 'compile' script.
  4. run the 'install' script to install the executable, plugins and docs.
    • the executable 'qtstalker' will be copied to the ~/Qtstalker directory
    • the plugins will be copied to ~/Qtstalker/plugins directory
    • the docs will be copied to ~/Qtstalker/docs directory
  5. Thats it.
