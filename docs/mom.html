Momentum - MOM


Parameters:

  • Input - The data field the MOM is based on
  • Period - The MOM period
  • Smoothing - The MOM smoothing period
  • Smoothing Type - The type of MOM smoothing to use
  • Color - MOM color
  • Line Type - MOM line type
  • Label - The text to identify the MOM plot

Description:


Formula Reference:

Syntax
 MOM(Input, Smoothing Type, Period, Smoothing)
Function
 Input can be either an array of bars or a formula variable array of data.
Example
 MOM(Close, SMA, 10, 10)
