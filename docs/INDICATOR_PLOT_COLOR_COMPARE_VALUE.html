INDICATOR_PLOT_COLOR_COMPARE_VALUE


DESCRIPTION: Compares NAME and VALUE using OP. If the compare is TRUE, then COLOR is applied to NAME2.
SYNTAX: PLUGIN = INDICATOR_PLOT_COLOR_COMPARE_VALUE
NAME = <VARIABLE NAME>
OP = <OPERATOR>
VALUE = <DECIMAL>
NAME2 = <VARIABLE NAME>
COLOR = <COLOR>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=INDICATOR_PLOT_COLOR_COMPARE_VALUE,NAME=close,OP=GT,VALUE=0,NAME2=close,COLOR=green
