QUOTE_DATABASE_DUMP


DESCRIPTION: Dumps all the quotes from the database for an EXCHANGE:SYMBOL as a CSV text file. DIRECTORY will be the base directory tree that files will be stored. Each EXCHANGE will have its own directory containing the SYMBOL file.

The CSV format is EXCHANGE, SYMBOL, NAME, DATE, OPEN, HIGH, LOW, CLOSE, VOLUME, OI
SYNTAX: PLUGIN = QUOTE_DATABASE_DUMP
EXCHANGE = <EXCHANGE>
SYMBOL = <SYMBOL>
DIRECTORY = <DIRECTORY> - base directory to store files
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=QUOTE_DATABASE_DUMP, DIRECTORY=/home/me/csv, EXCHANGE=XNYS, SYMBOL=AAPL
