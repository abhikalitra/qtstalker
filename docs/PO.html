PO - Price Oscillator

There are 2 methods:

  • APO - Absolute Price Oscillator
  • PPO- Percentage Price Oscillator

APO

INDICATOR,PLUGIN,PO,APO,<NAME>,<INPUT>,<FAST PERIOD>,<SLOW PERIOD>,<MA TYPE>

  • <NAME> is the variable name given to the plot
  • <INPUT> is an input type
  • <FAST PERIOD> is an integer
  • <SLOW PERIOD> is an integer
  • <MA TYPE> MA type

PPO

INDICATOR,PLUGIN,PO,PPO,<NAME>,<INPUT>,<FAST PERIOD>,<SLOW PERIOD>,<MA TYPE>

  • <NAME> is the variable name given to the plot
  • <INPUT> is an input type
  • <FAST PERIOD> is an integer
  • <SLOW PERIOD> is an integer
  • <MA TYPE> MA type
