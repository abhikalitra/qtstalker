APO description

Script API


INDICATOR,PLUGIN,PO,APO,<NAME>,<INPUT>,<FAST PERIOD>,<SLOW PERIOD>,<MA TYPE>,<PLOT TYPE>,<COLOR>

INDICATOR,PLUGIN,PO,PPO,<NAME>,<INPUT>,<FAST PERIOD>,<SLOW PERIOD>,<MA TYPE>,<PLOT TYPE>,<COLOR>

  • <NAME> is the variable name given to the plot
  • <INPUT> is an input type
  • <FAST PERIOD> is an integer
  • <SLOW PERIOD> is an integer
  • <MA TYPE> MA type
  • <PLOT TYPE> is a plot type
  • <COLOR> is the color of the plot
