NORMALIZE - Normalize

Normalizes an array of values. Converts all values to scale between and including 0 and 100. This is useful when comparing data sets of varying value ranges e.g. the Dow Jones Industrial index (10000) and the S&P500 index (1000) and plotting them in a chart so the scaling is sane.



Script API



INDICATOR,PLUGIN,NORMALIZE,<NAME>,<INPUT>
  • <NAME> is the variable name given to the array
  • <INPUT> is an input type
