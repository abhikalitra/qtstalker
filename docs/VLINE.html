VLINE


DESCRIPTION: Displays a vertical line on an indicator.
SYNTAX: PLUGIN = VLINE
TYPE = <TYPE>
  • RO - read only, not user interactive, not saved to disk
  • RW - read write, user interactive, saved to disk
NAME = <VARIABLE NAME>
INDICATOR = <INDICATOR NAME>
EXCHANGE = <EXCHANGE>
SYMBOL = <SYMBOL>
DATE = <yyyy-MM-dd HH:mm:ss>
COLOR = <COLOR>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=VLINE, TYPE=RW, NAME=vline1, INDICATOR=Bars, EXCHANGE=XNYS, SYMBOL=GOOG, DATE=2011-02-08 16:00:00, COLOR=green
