Moving Average - MA



Parameters:

  • Input - The data field the MA is based on
  • Color - MA color
  • Line Type - MA line type
  • Label - The text to identify the MA plot
  • Period - The period used for the MA
  • MA Type - The type of MA to use
    • EMA - Exponential moving average
    • SMA - Simple moving average
    • WMA - Weighted moving average
    • Wilder - Wilder's moving average

Description:


Formula Reference:

Syntax
 MA(Input, MA Type, Period)
Function
 Input can be either an array of bars or a formula variable array of data.
Example
 MA(Close, SMA, 14)
