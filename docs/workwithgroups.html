Group Panel

Description:

The group tab allows you to create, edit and delete groups. A group is simply a collection of charts defined by the user. Above the groups is a combo box that contains a list of all groups. Use this drop down list to switch between groups


Right clicking with the mouse in the list will display the context menu. The functions are as follows.

New Group

Here you can create a new group. Selecting this will bring up a dialog where you are asked for a name to call your new group. The name must be unique, if not an error message will display.

Add To Group

Displays a dialog that allows you to add the selected group item to the group of your choice.

Delete Group Items

Opens a dialog that allows you to select one or more charts to remove from the group.


Delete Group

Deletes the current group and all of its contents.
