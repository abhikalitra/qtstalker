COLOR - Color Bars

This indicator is only used by scripts.


Script API

This function has 2 methods. The 'All' method will color all bars with the same color. The 'Compare' method compares the values of <INPUT> data array with the number <VALUE>. If they are equal then <COLOR> is applied to the <INPUT_2> data array.


All method:

INDICATOR,PLUGIN,COLOR,All,<INPUT>,<COLOR>
  • <INPUT> is the input type to color
  • <COLOR> is the color to use. Either a RGB value (#FF0000) or english word (red) can be used

The Compare method:

INDICATOR,PLUGIN,COLOR,Compare,<INPUT>,<INPUT 2>,<VALUE>,<COLOR>

  • <INPUT> is the input type that determines the color
  • <INPUT 2> is the input type that receives the color
  • <VALUE> is a decimal value compared in <INPUT>. If <VALUE> and <INPUT> are equal, then <COLOR> is applied to <INPUT 2>
  • <COLOR> is the color name that is applied to <INPUT 2>. The color can either be an RGB value (#FF0000) or english word (red)
