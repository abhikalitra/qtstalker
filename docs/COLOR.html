COLOR


DESCRIPTION: Compares NAME with NAME_2 using OP. If the result is TRUE, then COLOR is applied to NAME_3. An OFFSET is provided for each NAME so previous values can be used for comparison. The OFFSET is relative to the current position. e.g. an OFFSET value of 2 would use the value 2 positions previous, 0 would use the current posiiton.
SYNTAX: COMMAND = COLOR
STEP = <VARIABLE NAME>
NAME = <VARIABLE NAME>
NAME_OFFSET = <INTEGER>
OP = <OPERATOR>
NAME_2 = <VARIABLE NAME>
NAME_2_OFFSET = <INTEGER>
NAME_3 = <VARIABLE NAME>
NAME_3_OFFSET = <INTEGER>
COLOR = <COLOR>
RETURNS: 'OK' = Success
'ERROR' = Failure
EXAMPLE: COMMAND=COLOR;
STEP=color;
NAME=close;
NAME_OFFSET=0;
OP=GT;
NAME_2=close;
NAME_2_OFFSET=0;
NAME_3=close;
NAME_3_OFFSET=0;
COLOR=green
