COLOR


DESCRIPTION: Compares NAME and NAME2 using OP. If the compare is TRUE, then COLOR is applied to NAME3.
SYNTAX: PLUGIN = COLOR
NAME = <VARIABLE NAME>
OP = <OPERATOR>
NAME2 = <VARIABLE NAME>
NAME3 = <VARIABLE NAME>
COLOR = <COLOR>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=COLOR, NAME=close, OP=GT, NAME2=close, NAME3=close, COLOR=green
