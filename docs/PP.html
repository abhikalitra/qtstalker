PP - Pivot Point

Description



Script API


INDICATOR,PLUGIN,PP,<NAME>,<POINT>
  • <NAME> is the variable name given to the array
  • <POINT> is a value between 0 and 5
    • 0 = first resistance
    • 1 = second resistance
    • 2 = third resistance
    • 3 = first support
    • 4 = second support
    • 5 = third support
