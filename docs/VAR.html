VAR - Variance

See Description

Variance is basically a standard deviation calculation except the last step which is to apply a square root to the variance number.

VAR

INDICATOR,PLUGIN,VAR,<NAME>,<INPUT>,<PERIOD>

  • <NAME> is the variable name given to the array
  • <INPUT> is an input type
  • <PERIOD> is an integer
