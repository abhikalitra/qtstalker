VAR


DESCRIPTION: Variance is a standard deviation calculation except the last step which is to apply a square root to the variance number.
SYNTAX: COMMAND = VAR
STEP = <VARIABLE NAME>
NAME = <VARIABLE NAME>
INPUT = <VARIABLE NAME>
PERIOD = <INTEGER>
DEVIATION = <DECIMAL>
RETURNS: 'OK' = Success
'ERROR' = Failure
EXAMPLE: COMMAND=VAR;
STEP=var;
NAME=var;
INPUT=close;
PERIOD=10;
DEVIATION=2
