VBP


DESCRIPTION: Volume By Price.

SYNTAX: PLUGIN = VBP
NAME = <VARIABLE NAME>
INPUT_CLOSE = <VARIABLE NAME>
INPUT_VOLUME = <VARIABLE NAME>
COLOR_UP = <COLOR>
COLOR_DOWN = <COLOR>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=VBP, NAME=vbp, INPUT_CLOSE=Close, INPUT_VOLUME=Volume, COLOR_UP=green, COLOR_DOWN=red
