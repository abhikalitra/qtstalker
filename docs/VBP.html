VBP


DESCRIPTION: Volume By Price.

SYNTAX: COMMAND = VBP
STEP = <VARIABLE NAME>
NAME = <VARIABLE NAME>
CLOSE = <VARIABLE NAME>
VOLUME = <VARIABLE NAME>
COLOR_UP = <COLOR>
COLOR_DOWN = <COLOR>
RETURNS: 'OK' = Success
'ERROR' = Failure
EXAMPLE: COMMAND=VBP;
STEP=vbp;
NAME=vbp;
CLOSE=Close;
VOLUME=Volume;
COLOR_UP=green;
COLOR_DOWN=red
