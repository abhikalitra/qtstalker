Format:

INDICATOR,PLUGIN,BETA,<NAME>,<INPUT 1>,<INPUT 2>,<PERIOD>

  • <INPUT 1> is the index to compare to. ie S&P 500
  • <INPUT 2> is the symbol to compare with <INPUT 1>
  • <PERIOD> is an integer value for period
