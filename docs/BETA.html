BETA - BETA Coefficient


DESCRIPTION:    BETA

SYNTAX:             PLUGIN = BETA
                           NAME = <VARIABLE NAME>
                           INPUT = <VARIABLE NAME> Security to compare.
                           INPUT2 = <VARIABLE NAME> Index or benchmark used for deviation comparison such as the S&P500.
                           PERIOD = <INTEGER>

RETURNS:           '0' = Success
                          'ERROR' = Failure

EXAMPLE:           PLUGIN=BETA,NAME=beta,INPUT=close,INPUT2=sp500,PERIOD=10
