BuyArrow

Description:

A small up pointing arrow. Commonly used to indicate a buy point on a chart.



To create a BuyArrow, select draw mode from the main toolbar and right click with the mouse on the chart you wish to apply it to. This can be either the main chart area or any of the indicators that appear in the indicator chart area. Select "New Chart Object" from the popup menu and select BuyArrow from the menu.

After selecting the menu option, the mouse pointer will change to a hand pointer. To place the BuyArrow on the chart the user must click the left mouse button. The object will then appear on the chart.

Selecting:

In order to edit the BuyArrow, the user must first select it. First, the user must be in draw mode. Select the BuyArrow by left clicking with the mouse anywhere on the object. The object will be selected when a "grab box" appears just above the BuyArrow like this:



You then can right click the mouse for a context menu for the object. You may also double-left click on the object to bring up the edit dialog directly. There are three options when a BuyArrow is selected. Edit, Move and Delete.

Edit:

The buy arrow settings are defined as follows:

  • Color - The color of the arrow.
  • Value - The value placement on the chart.
  • Set Default - If checked, any new BuyArrows will use the current settings as the default.


Move:

To drag the BuyArrow across the chart, click the left mouse button on the "grab box". The BuyArrow will then follow the mouse pointer. To drop the BuyArrow, click the left mouse button.

Delete:

This will permanently delete the BuyArrow from the chart.
