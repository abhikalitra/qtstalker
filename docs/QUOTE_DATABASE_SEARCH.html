QUOTE_DATABASE SEARCH


DESCRIPTION: Searches the database for EXCHANGE and SYMBOL pattern and returns a semicolon delimited string of symbols that match. Result is stored in KEY.
SYNTAX: PLUGIN = QUOTE_DATABASE
METHOD = SEARCH
EXCHANGE = <EXCHANGE PATTERN>
  1. % for partial match
  2. * for anything
SYMBOL = <SYMBOL PATTERN>
  1. % for partial match
  2. * for anything
RETURNS: '0' = Success
'ERROR' = Failure
KEY = QUOTE_DATABASE_SEARCH_SYMBOLS
EXAMPLE: PLUGIN=QUOTE_DATABASE, METHOD=SEARCH, EXCHANGE=*, SYMBOL=%OO%
