CANDLES - Candlesticks


DESCRIPTION:    The standard Japanese candlestick plot. Colors are relative to the previous Close. COLOR_UP is a higher close than previous Close. COLOR_DOWN is a lower Close than the                            previous Close. COLOR_NEUTRAL is no change from the previous Close.

SYNTAX:             PLUGIN = CANDLES
                           NAME = <VARIABLE NAME>
                           COLOR_UP = <COLOR>
                           COLOR_DOWN = <COLOR>
                           COLOR_NEUTRAL = <COLOR>

RETURNS:           '0' = Success
                          'ERROR' = Failure

EXAMPLE:           PLUGIN=CANDLES,NAME=candles,COLOR_UP=green,COLOR_DOWN=red,COLOR_NEUTRAL=blue
