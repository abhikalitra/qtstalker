SELECT_DIALOG


DESCRIPTION: Displays a dialog that allows ITEMS to be selected from a list. The selected ITEMS are stored in KEY as a semicolon delimited string.
SYNTAX: PLUGIN = SELECT_DIALOG
MODE = <BOOLEAN>
  1. 0 - Multi selection
  2. 1 - Single selection
TITLE = <LIST CAPTION>
ITEMS = <ITEMS> semicolon delimited string
RETURNS: '0' = Success
'ERROR' = Failure
KEY = SELECT_DIALOG_SELECTED
EXAMPLE: PLUGIN=SELECT_DIALOG,ITEMS=item1;item2;item3
