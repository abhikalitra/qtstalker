COMPARE


DESCRIPTION: Compares NAME2 and NAME3 using OP. If the compare is TRUE, then NAME = NAME4, else NAME = NAME5.
SYNTAX: PLUGIN = COMPARE
NAME = <VARIABLE NAME>
OP = <OPERATOR>
NAME2 = <VARIABLE NAME>
NAME3 = <VARIABLE NAME> - can be an array or decimal constant value
NAME4 = <VARIABLE NAME> - can be an array or decimal constant value
NAME5 = <VARIABLE NAME> - can be an array or decimal constant value
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=COMPARE, NAME=comp, NAME2=Close, OP=GT, NAME3=Open, NAME4=1, NAME5=0
