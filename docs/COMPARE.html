COMPARE


DESCRIPTION: Compares INPUT with INPUT_2 using OP. If the result is TRUE, then NAME = INPUT_3, else NAME = INPUT_4. An OFFSET is provided for each INPUT so previous values can be used for comparison. The OFFSET is relative to the current position. e.g. an OFFSET value of 2 would use the value 2 positions previous, 0 would use the current posiiton. Decimal constant values can be used for INPUT_2, INPUT_3, INPUT_4 e.g. '24.627'
SYNTAX: COMMAND = COMPARE
STEP = <VARIABLE NAME>
NAME = <VARIABLE NAME>
OP = <OPERATOR>
INPUT = <VARIABLE NAME>
OFFSET = <INTEGER>
INPUT_2 = <VARIABLE NAME> - can be an array or decimal constant value
OFFSET_2 = <INTEGER>
INPUT_3 = <VARIABLE NAME> - can be an array or decimal constant value
OFFSET_3 = <INTEGER>
INPUT_4 = <VARIABLE NAME> - can be an array or decimal constant value
OFFSET_4 = <INTEGER>
RETURNS: 'OK' = Success
'ERROR' = Failure
EXAMPLE: COMMAND=COMPARE;
STEP=compare1;
NAME=comp;
INPUT=Close;
OFFSET=0;
OP=GT;
INPUT_2=Open;
OFFSET_2=0;
INPUT_3=1;
OFFSET_3=0;
INPUT_4=0
OFFSET_4=0;
