COMPARE - Compare Two Variables

This function is only used in scripts.


Script API

Compares  <INPUT> with <INPUT 2> using the <OPERATOR> for comparison. <INPUT> and <INPUT 2> can either be a data array or a numerical value. The result is a boolean array of TRUE / FALSE values. The comparison order is made <INPUT> <OPERATOR> <INPUT 2>.

Format:

INDICATOR,PLUGIN,COMPARE,<NAME>,<INPUT>,<INPUT 2>,<OPERATOR>
  • <NAME> is the variable name given to the array
  • <INPUT> can be an input type or a decimal value
  • <INPUT 2> can be an input type or a decimal value
  • <OPERATOR> is an operator type
