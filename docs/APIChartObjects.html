Script API - Chart Objects

These CUS commands allow creation and manipulation of chart objects applied to scripted indicators.

CO,Buy,<EXCHANGE>,<SYMBOL>,<INDICATOR>,<DATE>,<PRICE>,<COLOR>

Create a new Buy chart object.

  • <EXCHANGE> is an existing valid exchange code
  • <SYMBOL> is an existing valid security symbol
  • <INDICATOR> is an existing indicator
  • <DATE> is the Qt::ISODate format to position the chart object on the indicator.
  • <PRICE> is the price position on the indicator
  • <COLOR> is the chart object color

On success, a '0' is returned. Failure returns 'ERROR'.

CO,Retracement,<EXCHANGE>,<SYMBOL>,<INDICATOR>,<START DATE>,<END DATE>,<HIGH>,<LOW>,<PERCENTAGE>,<COLOR>

Create a new Retracement chart object.

  • <EXCHANGE> is an existing valid exchange code
  • <SYMBOL> is an existing valid security symbol
  • <INDICATOR> is an existing indicator
  • <START DATE> is the Qt::ISODate format to position the chart object on the indicator
  • <END DATE> is the Qt::ISODate format to position the chart object on the indicator
  • <HIGH> is the high price position on the indicator
  • <LOW> is the low price position on the indicator
  • <PERCENTAGE> is the percentage retracement eg. 0.5 would be a 50% retracement, 1.0 would be 100% retracement
  • <COLOR> is the chart object color

On success, a '0' is returned. Failure returns 'ERROR'.
