QUOTE_DATABASE_RENAME


DESCRIPTION: Rename a symbol in the database.
SYNTAX: PLUGIN = QUOTE_DATABASE_RENAME
OLD_EXCHANGE = <EXCHANGE>
OLD_SYMBOL = <SYMBOL>
NEW_EXCHANGE = <EXCHANGE>
NEW_SYMBOL = <SYMBOL>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=QUOTE_DATABASE_RENAME,OLD_EXCHANGE=XNAS,OLD_SYMBOL=GOOG,NEW_EXCHANGE=XNYS,NEW_SYMBOL=GOOG
