CHART_OBJECT_RETRACEMENT_TEMP


DESCRIPTION: Creates a retracement chart object. This is a read only version that is non-interactive when displayed and is not saved to disk.
SYNTAX: PLUGIN = CHART_OBJECT_RETRACEMENT_TEMP
DATE = <yyyy-MM-dd HH:mm:ss>
DATE2 = <yyyy-MM-dd HH:mm:ss>
COLOR = <COLOR>
HIGH = <DECIMAL>
LOW = <DECIMAL>
LINE1 = <DECIMAL>
LINE2 = <DECIMAL>
LINE3 = <DECIMAL>
LINE4 = <DECIMAL>
LINE5 = <DECIMAL>
LINE6 = <DECIMAL>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=CHART_OBJECT_RETRACEMENT_TEMP,DATE=2011-01-01 09:30:00,DATE2=2011-02-08 16:00:00,COLOR=red,HIGH=500,LOW=400,LINE1=0.236,LINE2=0.381,LINE3=0.618
