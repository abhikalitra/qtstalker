CHART_OBJECT_TEXT


DESCRIPTION: Displays text on an indicator.
SYNTAX: COMMAND = CHART_OBJECT_TEXT
STEP = <VARIABLE NAME>
DATE = <yyyy-MM-dd HH:mm:ss>
PRICE = <DECIMAL>
COLOR = <COLOR>
LABEL = <TEXT>
Z = <INTEGER>
RETURNS: 'OK' = Success
'ERROR' = Failure
EXAMPLE: COMMAND=CHART_OBJECT_TEXT;
STEP=text1;
DATE=2011-02-08 16:00:00;
PRICE=450.50;
COLOR=red;
LABEL=Look Here;
Z=1
