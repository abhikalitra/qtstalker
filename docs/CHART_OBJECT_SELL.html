CHART_OBJECT_SELL


DESCRIPTION: Displays an downward pointing arrow on an indicator. Used to indicate a sell point.

SYNTAX: COMMAND = CHART_OBJECT_SELL
STEP = <VARIABLE NAME>
DATE = <yyyy-MM-dd HH:mm:ss>
PRICE = <DECIMAL>
COLOR = <COLOR>
Z = <INTEGER>
RETURNS: 'OK' = Success
'ERROR' = Failure
EXAMPLE: COMMAND=CHART_OBJECT_SELL;
STEP=sell1;
DATE=2011-02-08 16:00:00;
PRICE=450.50;
COLOR=red;
Z=1;
