WILDER_MA


DESCRIPTION: J. Welles Wilder introduced this moving average calculation in his 1978 book: New Concepts in Technical Trading Systems. It produces a somewhat different result than an EMA. He used this calculation for all of his indicators. So, if you want to stay true to his methods, you may use this calculation instead of the standard EMA type.
SYNTAX: PLUGIN = WILDER_MA
NAME = <VARIABLE NAME>
INPUT = <VARIABLE NAME>
PERIOD = <INTEGER>
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=WILDER_MA,NAME=wilderma,INPUT=close,PERIOD=14
