CSV - Independent data loading for Comma-Separated Values text files

'CSV' is a companion application for Qtstalker to "Import CSV Quotes". It enables the loading of data to the SQLite database for users that do not have better independent techniques for data management.

Note that this application will always be basic, as it is intended that you manage your data as a separate concern.

Obtain the data as CSV files

If you purchase data, then it should be available as Comma-Separated Values text files.

If you want to use free end-of-day data (e.g. provided by Yahoo) then you need to obtain that each day as a CSV file using some independent means, e.g. manual download via web browser; or automated using 'wget', etc.

The field delimiter can be any character. The default is comma.

Data quality

You should have some independent means for data verification, e.g a Perl script.

This "CSV" application will do some basic error detection while loading the data.

File names

The data filenames can have any filename extension, e.g. ibm.txt or ibm.csv or any other.

It is useful to set the "Use file name for symbol" option. Some data files will not have the symbol name on every data row. If they do, then specify that field to be "Ignore".

The filename extension will be stripped and the basename part will be used, e.g. for "ibm.txt" the symbol will be "ibm". Similarly for uppercase, e.g. for "GOOG.csv" the symbol will be "GOOG".

Date or Date-Time format

The date format is using the Qt date format mask. That documentation explains how to specify any date or date-time format.

Some examples:

Input Format
2009-07-15 yyyy-MM-dd
15Jul09 ddMMMyy
2009/07/15 16:42:30 yyyy/MM/dd HH:mm:ss

Operation

Double-click an item in the "Rule" column to show the edit dialog.

Double-click in the "Status" column will activate that rule once.

Setting the time interval to greater than zero on a rule will set that rule to active for as long as the CSV application is running.

Setting the timer to zero will set the status to inactive (i.e. it will run once, only when activated).
