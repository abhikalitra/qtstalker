QUOTE_DATABASE SET


DESCRIPTION: Stores a quote into the database.
SYNTAX: PLUGIN = QUOTE_DATABASE
METHOD = SET
EXCHANGE = <EXCHANGE>
SYMBOL = <SYMBOL>
DATE_FORMAT = <DATE FORMAT>
DATE = <DATE>
OPEN = <DECIMAL>
HIGH = <DECIMAL>
LOW = <DECIMAL>
CLOSE = <DECIMAL>
VOLUME = <INTEGER>
INTEREST = <INTEGER> *Optional usage.
NAME = <COMPANY NAME> *Optional usage
TYPE = <SECURITY TYPE> *Optional usage. User defined, defaults to 'Stock'
SAVE = <BOOLEAN> Saves quotes. After all quotes have been set.
RETURNS: '0' = Success
'ERROR' = Failure
EXAMPLE: PLUGIN=QUOTE_DATABASE, METHOD=SET, EXCHANGE=XNYS, SYMBOL=GOOG, DATE_FORMAT=yyyy-MM-dd HH:mm:ss, DATE=2011-02-11 16:00:00, OPEN=500, HIGH=525, LOW=475, CLOSE=510, VOLUME=123456789

PLUGIN=QUOTE_DATABASE, METHOD=SET, SAVE=1
