VIDYA - Variable Index Dynamic Average

Description



Script API


INDICATOR,PLUGIN,VIDYA,<NAME>,<INPUT>,<PERIOD>,<VOLUME PERIOD>,<PLOT TYPE>,<COLOR>

  • <NAME> is the variable name given to the array
  • <INPUT> is an input type
  • <PERIOD> is an integer ( 14 is suggested)
  • <VOLUME_PERIOD>  is an integer ( 10 is suggested)
  • <PLOT TYPE> is a plot type
  • <COLOR> is the color of the plot
